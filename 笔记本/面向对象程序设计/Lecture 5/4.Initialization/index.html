    Initializer list¶

        Class Point {
    private:
        const float x, y; // жҲҗе‘ҳеҸҳйҮҸжҳҜ const
    public:
        // жһ„йҖ еҮҪж•°дҪҝз”ЁеҲқе§ӢеҢ–еҲ—иЎЁ
        Point(float xa, float ya) : y(ya), x(xa) {} // еҲқе§ӢеҢ–еҲ—иЎЁдёӯ y еңЁ x д№ӢеүҚ
};
    еңЁиҝҷдёӘзӨәдҫӢдёӯпјҢPoint зұ»зҡ„жҲҗе‘ҳ x е’Ң y иў«еЈ°жҳҺдёә constгҖӮиҝҷдҪҝеҫ— е®ғд»¬еҝ…йЎ»еңЁеҜ№иұЎжһ„йҖ ж—¶пјҲеҚіжһ„йҖ еҮҪж•°дҪ“жү§иЎҢд№ӢеүҚпјүиў«еҲқе§ӢеҢ–пјҢиҖҢдёҚиғҪеңЁжһ„йҖ еҮҪж•°дҪ“еҶ…иөӢеҖјгҖӮ

    еҲқе§ӢеҢ–еҲ—иЎЁзҡ„е…ій”®зү№жҖ§¶

    • Can initialize any type of data

      • еҲқе§ӢеҢ–еҲ—иЎЁеҸҜд»Ҙз”ЁжқҘеҲқе§ӢеҢ–д»»дҪ•зұ»еһӢзҡ„жҲҗе‘ҳж•°жҚ®гҖӮ
      • pseudo-constructor calls for built-insпјҡеҜ№дәҺеҶ…зҪ®зұ»еһӢпјҲеҰӮ int, floatпјүпјҢе®ғжү§иЎҢзҡ„жҳҜзұ»дјјжһ„йҖ еҮҪж•°зҡ„еҲқе§ӢеҢ–иЎҢдёәгҖӮ
      • No need to perform assignment within body of ctorпјҡдҪҝз”ЁеҲқе§ӢеҢ–еҲ—иЎЁжҳҜеҲқе§ӢеҢ–пјҢиҖҢдёҚжҳҜиөӢеҖјгҖӮе®ғеңЁжһ„йҖ еҮҪж•°дҪ“жү§иЎҢд№ӢеүҚе®ҢжҲҗпјҢеӣ жӯӨж•ҲзҺҮжӣҙй«ҳпјҢд№ҹжҳҜеҲқе§ӢеҢ– const жҲҗе‘ҳгҖҒеј•з”ЁжҲҗе‘ҳе’Ңеҹәзұ»жҲҗе‘ҳзҡ„е”ҜдёҖж–№жі•гҖӮ

    • Order of initialization is order of declaration

      • еҲқе§ӢеҢ–йЎәеәҸз”ұеЈ°жҳҺеҶіе®ҡпјҡ иҝҷжҳҜеҲқе§ӢеҢ–еҲ—иЎЁжңҖйҮҚиҰҒзҡ„и§„еҲҷгҖӮ
      • Not the order in the initializer list!пјҡеңЁд»Јз ҒзӨәдҫӢдёӯпјҢy(ya) еңЁ x(xa) д№ӢеүҚпјҢдҪҶеҰӮжһң x еңЁзұ»е®ҡд№үдёӯе…ҲеЈ°жҳҺпјҢйӮЈд№Ҳ x дјҡе…ҲдәҺ y еҲқе§ӢеҢ–гҖӮ
      • Destroyed in the reverse order.пјҡжҲҗе‘ҳзҡ„й”ҖжҜҒйЎәеәҸдёҺе®ғд»¬зҡ„еҲқе§ӢеҢ–йЎәеәҸзӣёеҸҚгҖӮеӣ жӯӨе…Ҳй”ҖжҜҒ y еҶҚй”ҖжҜҒ x гҖӮ

    жҖ»з»“¶

    еҲқе§ӢеҢ–еҲ—иЎЁжҳҜ C++ дёӯе®һзҺ°дҝқиҜҒеҲқе§ӢеҢ– зҡ„жҺЁиҚҗж–№ејҸгҖӮе®ғдёҚд»…жҳҜеҲқе§ӢеҢ– const жҲҗе‘ҳе’Ңеј•з”ЁжҲҗе‘ҳзҡ„еҝ…йңҖж–№ејҸпјҢд№ҹзЎ®дҝқдәҶжҲҗе‘ҳжҢүз…§еЈ°жҳҺйЎәеәҸиҝӣиЎҢеҲқе§ӢеҢ–пјҢд»ҺиҖҢйҒҝе…ҚжҪңеңЁзҡ„дҫқиө–й”ҷиҜҜгҖӮ

    : y(ya), x(xa) {}жҳҜд»Җд№Ҳж„ҸжҖқпјҹ

    : y(ya), x(xa) {}жҳҜ C++ жһ„йҖ еҮҪж•°зҡ„еҲқе§ӢеҢ–еҲ—иЎЁ (Initializer List) йғЁеҲҶпјҢз”ЁдәҺеңЁжһ„йҖ еҮҪж•°дҪ“жү§иЎҢд№ӢеүҚпјҢеҜ№зұ»зҡ„жҲҗе‘ҳиҝӣиЎҢеҲқе§ӢеҢ–гҖӮ

    дҪңз”Ёе’ҢиҜӯжі•пјҡ¶

    1. еҶ’еҸ· (:)пјҡ ж Үеҝ—зқҖжһ„йҖ еҮҪж•°еӨҙпјҲеҸӮж•°еҲ—иЎЁпјүзҡ„з»“жқҹе’ҢеҲқе§ӢеҢ–еҲ—иЎЁзҡ„ејҖе§ӢгҖӮ

    2. y(ya), x(xa)пјҡ иҝҷжҳҜеҲқе§ӢеҢ–еҲ—иЎЁзҡ„еҶ…е®№гҖӮе®ғзҡ„ж јејҸжҳҜ жҲҗе‘ҳеҸҳйҮҸеҗҚ(еҸӮж•°еҗҚ)гҖӮ

      • е®ғиЎЁзӨәе°Ҷжһ„йҖ еҮҪж•°еҸӮж•° ya зҡ„еҖјз”ЁжқҘеҲқе§ӢеҢ–жҲҗе‘ҳеҸҳйҮҸ yгҖӮ
      • е°Ҷжһ„йҖ еҮҪж•°еҸӮж•° xa зҡ„еҖјз”ЁжқҘеҲқе§ӢеҢ–жҲҗе‘ҳеҸҳйҮҸ xгҖӮ

    3. иҠұжӢ¬еҸ· ({})пјҡ иҝҷжҳҜжһ„йҖ еҮҪж•°зҡ„еҮҪж•°дҪ“гҖӮеңЁеҲқе§ӢеҢ–еҲ—иЎЁе®ҢжҲҗеҗҺпјҢжүҚдјҡжү§иЎҢиҠұжӢ¬еҸ·еҶ…йғЁзҡ„д»Јз ҒгҖӮ

    дёәд»Җд№ҲдҪҝз”ЁеҲқе§ӢеҢ–еҲ—иЎЁпјҹ¶

    1. дҝқиҜҒеҲқе§ӢеҢ–пјҡ е®ғжҳҜ C++ е®ҳж–№жҺЁиҚҗзҡ„еҲқе§ӢеҢ–жүҖжңүжҲҗе‘ҳзҡ„ж–№ејҸгҖӮ
    2. еҝ…йңҖжҖ§пјҡ еҜ№дәҺ const жҲҗе‘ҳпјҲеҰӮжӮЁзӨәдҫӢдёӯзҡ„ x, yпјүе’Ңеј•з”ЁжҲҗе‘ҳпјҢе®ғд»¬еңЁеҲӣе»әеҗҺдёҚиғҪиў«иөӢеҖјпјҢжүҖд»Ҙе®ғд»¬еҝ…йЎ»йҖҡиҝҮеҲқе§ӢеҢ–еҲ—иЎЁиҝӣиЎҢеҲқе§ӢеҢ–гҖӮ
    3. ж•ҲзҺҮпјҡ зӣёжҜ”еңЁиҠұжӢ¬еҸ·еҶ…дҪҝз”Ё = иҝӣиЎҢиөӢеҖјпјҢеҲқе§ӢеҢ–еҲ—иЎЁжҳҜеҲқе§ӢеҢ–пјҲзӣҙжҺҘжһ„е»әпјүпјҢиҖҢдёҚжҳҜе…Ҳй»ҳи®Өжһ„йҖ еҶҚиөӢеҖјпјҢж•ҲзҺҮжӣҙй«ҳгҖӮ

    дёәд»Җд№ҲжІЎжңүз”ЁеҲҶеҸ· (;) з»“е°ҫпјҹ

    еңЁ C++ дёӯпјҢеҲҶеҸ· (;) з”ЁдәҺз»“жқҹдёҖдёӘиҜӯеҸҘ (Statement)гҖӮиҖҢ Point(float xa, float ya) : y(ya), x(xa) {} жҳҜдёҖж®өеҮҪж•°пјҲжҲ–жһ„йҖ еҮҪж•°пјүзҡ„е®ҡд№үпјҢиҖҢдёҚжҳҜдёҖдёӘжҷ®йҖҡзҡ„иҜӯеҸҘгҖӮ

    • еҮҪж•°е®ҡд№үз»“жһ„пјҡ C++ зҡ„еҮҪж•°е’Ңжһ„йҖ еҮҪж•°е®ҡд№үжҳҜз”ұеҮҪж•°еӨҙе’ҢеҮҪж•°дҪ“з»„жҲҗзҡ„гҖӮ

      • еҮҪж•°еӨҙпјҡ Point(float xa, float ya)
      • еҲқе§ӢеҢ–еҲ—иЎЁпјҡ : y(ya), x(xa)
      • еҮҪж•°дҪ“пјҡ {}

    • и§„еҲҷпјҡ еҮҪж•°зҡ„е®ҡд№үпјҲжҸҗдҫӣеҮҪж•°дҪ“жҲ–иҠұжӢ¬еҸ· {} зҡ„е®Ңж•ҙе®һзҺ°пјүеҗҺйқўдёҚеҠ еҲҶеҸ·гҖӮ

    • еҜ№жҜ”пјҡ еҸӘжңүеҮҪж•°еЈ°жҳҺпјҲеҺҹеһӢпјҢеҚіеҸӘе‘ҠиҜүзј–иҜ‘еҷЁеҮҪж•°зӯҫеҗҚпјҢдёҚжҸҗдҫӣеҮҪж•°дҪ“пјүж—¶жүҚйңҖиҰҒеҲҶеҸ·пјҢдҫӢеҰӮеңЁеӨҙж–Үд»¶ .h дёӯ: Point(float xa, float ya);дҪҶеңЁ.cpp ж–Үд»¶дёӯпјҲжҲ–еңЁзұ»е®ҡд№үдёӯзӣҙжҺҘжҸҗдҫӣеҮҪж•°дҪ“ж—¶пјүпјҢе®Ңж•ҙзҡ„е®ҡд№үдёҚйңҖиҰҒеҲҶеҸ·гҖӮ

    жҖ»з»“пјҡ¶

    дёҚиғҪеңЁзұ»еҶ…йғЁж”ҫдёҖдёӘеҮҪж•°еЈ°жҳҺеҗҺйқўи·ҹзқҖдёҖдёӘеҮҪж•°е®ҡд№үпјҢC++ зј–иҜ‘еҷЁи®Өдёәпјҡ

    • Point(float xa, float ya); \(\rightarrow\) иҝҷжҳҜдёҖдёӘ еЈ°жҳҺ пјҢеҗҺйқўйңҖиҰҒеҲҶеҸ·гҖӮ
    • Point(float xa, float ya) : y(ya), x(xa) {} \(\rightarrow\) иҝҷжҳҜдёҖдёӘе®ҡд№үпјҢеӣ дёәе®ғжҸҗдҫӣдәҶе®һзҺ°пјҲеҢ…жӢ¬еҲқе§ӢеҢ–еҲ—иЎЁе’ҢеҮҪж•°дҪ“пјүпјҢеҗҺйқўдёҚйңҖиҰҒеҲҶеҸ·гҖӮ

    1. A default argument is a value given in the declaration that the compiler automatically inserts if you don't provide a value in the function call.

      • е®ҡд№үпјҡ й»ҳи®ӨеҸӮж•°жҳҜеңЁеҮҪж•°еЈ°жҳҺж—¶иөӢдәҲеҸӮж•°зҡ„дёҖдёӘеҖјгҖӮ

    Constant objects¶

    1. д»Җд№ҲжҳҜеёёйҮҸеҜ№иұЎпјҹ

      • зӨәдҫӢпјҡ const Currency the_raise(42, 38);
      • е®ҡд№үпјҡ еҪ“дёҖдёӘеҜ№иұЎдҪҝз”Ё const е…ій”®еӯ—еЈ°жҳҺж—¶пјҢе®ғе°ұжҲҗдёәдёҖдёӘеёёйҮҸеҜ№иұЎгҖӮ
      • еҗ«д№үпјҡ const ж„Ҹе‘ізқҖиҜҘеҜ№иұЎзҡ„зҠ¶жҖҒпјҲеҚіе®ғзҡ„йқһйқҷжҖҒжҲҗе‘ҳеҸҳйҮҸ/еӯ—ж®өпјүеңЁеҜ№иұЎиў«еҲӣе»әеҗҺпјҢдёҚиғҪиў«дҝ®ж”№гҖӮ

    2. еёёйҮҸеҜ№иұЎеёҰжқҘзҡ„й—®йўҳ

      • What if an object is const?

        • еҰӮжһңдёҖдёӘеҜ№иұЎжҳҜ constпјҢйӮЈд№Ҳд»»дҪ•иҜ•еӣҫдҝ®ж”№е…¶еҶ…йғЁзҠ¶жҖҒпјҲеӯ—ж®өпјүзҡ„д»Јз ҒйғҪе°ҶжҳҜ**йқһжі•**зҡ„гҖӮ

      • What member functions can access the internals?

        • ж ёеҝғй—®йўҳпјҡ ж—ўз„¶еёёйҮҸеҜ№иұЎдёҚиғҪиў«дҝ®ж”№пјҢйӮЈд№Ҳе“ӘдәӣжҲҗе‘ҳеҮҪж•°еҸҜд»Ҙиў«и°ғз”Ёпјҹ
        • зӯ”жЎҲпјҲC++и§„еҲҷпјүпјҡ еҸӘжңүйӮЈдәӣиў«еЈ°жҳҺдёә const жҲҗе‘ҳеҮҪж•°зҡ„ж–№жі•жүҚиғҪиў«еёёйҮҸеҜ№иұЎи°ғз”ЁгҖӮ
          • const жҲҗе‘ҳеҮҪж•°пјҡжүҝиҜәдёҚдјҡдҝ®ж”№еҜ№иұЎзҡ„зҠ¶жҖҒгҖӮзј–иҜ‘еҷЁдјҡејәеҲ¶жү§иЎҢиҝҷдёӘжүҝиҜәгҖӮ
          • йқһ const жҲҗе‘ҳеҮҪж•°пјҡиў«и®Өдёәе…·жңүдҝ®ж”№еҜ№иұЎзҠ¶жҖҒзҡ„жҪңеҠӣпјҢеӣ жӯӨдёҚиғҪиў« const еҜ№иұЎи°ғз”ЁгҖӮ

    жҖ»з»“¶

    еёёйҮҸеҜ№иұЎпјҲеҰӮ the_raiseпјүжҳҜ C++ дёӯе®һзҺ°дёҚеҸҳжҖ§пјҲImmutabilityпјүе’ҢеҘ‘зәҰзҡ„йҮҚиҰҒе·Ҙе…·гҖӮйҖҡиҝҮе°ҶеҜ№иұЎеЈ°жҳҺдёә constпјҢзЁӢеәҸе‘ҳеҗ‘зј–иҜ‘еҷЁдҝқиҜҒиҜҘеҜ№иұЎзҡ„зҠ¶жҖҒе°ҶдҝқжҢҒдёҚеҸҳпјҢиҝҷжңүеҠ©дәҺд»Јз Ғе®үе…ЁжҖ§е’ҢеӨҡзәҝзЁӢзј–зЁӢгҖӮеҸӘжңүйӮЈдәӣжҳҺзЎ®жүҝиҜәдёҚдјҡдҝ®ж”№еҜ№иұЎзҡ„жҲҗе‘ҳеҮҪж•°пјҲеҚі const ж–№жі•пјүжүҚиў«е…Ғи®ёеңЁеёёйҮҸеҜ№иұЎдёҠж“ҚдҪңгҖӮ

    Example

        #include<iostream>
    using namespace std;
    class Y{
        int m_i;
        public:
        Y (int i){ //жҺҘеҸ—дёҖдёӘiдҪңдёәжһ„йҖ 
            m_i = i;
            cout << "In construction " << endl;
        }
        Y (Y &next){ //жҺҘеҸ—дёҖдёӘYдҪңдёәжһ„йҖ 
            m_i = next.m_i;
            cout << "In copy construction " << endl;
        }
    };
    class Myclass{
        int a;
        double b;
        Y m_y;
        public:
        Myclass(int x, double y, Y m): a(x), b(y), m_y(m){
            m_y = m;
            cout << a << " " << b << endl;
        }

        Myclass(int x, double y, int m): a(x), b(y), m_y(m){
            cout << a << " " << b << endl;
        }
    };

    int main(){
        char a[4];
        char *b = new char[4];

        cout << a << endl;
        cout << " " << endl;
        cout << b << endl;

        Myclass(3, 4, Y(1));
        Myclass(3, 5, 6);

        return 1;
    }
