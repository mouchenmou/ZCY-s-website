    我的想法：¶

    我现在是这么想的，在StronglyConnectedComponents这个函数中，先根据G的NumOfEdges得出总共有多少边，然后构建一个数组一共 有NumOfEdges行，3列，命名为edge，第一列用于存放起始顶点，第二列用于存放终止顶点，比如一条边是从节点1指向节点3的那么第 一列就存1，第二列就存3，第三列用于判断这条边是否参与构建DFS spanning tree（后面会讲），没参与就记为0（也就是back edge），参与了就记为1，因此初始化的时候全都是0。构建完edge这个数组之后，还要构建一个数组，命名DFS_Tree，一共有NumOfVe rtices列，3行。第一行用于存放它在构建这棵DFS Spanning Tree时时第几个被加入的元素，也就是number，如果是第一个就记作1， 第二个就记作2，所有的number初始化都为0，第二行是low，用于存放每一个顶点最高能跳转到哪里（后面我会定义一个名为Find_low 的函数来计算low），这个low的计算方式就是min{当前节点的number，当前节点如果有back edge的话，这个back edge能跳到的最小的number，当前节点的子树中所有的back edge能跳转到的最小的number}，所有的low初始化也都为0。第三行用于记录每一个节点在这个Spanning Tree中的parent，初始化全都为-1。构建完这个数组之后，就需要开始找DFS Spanning Tree了，不如就以顶点0为起点构建Spanning Tree，先定义一个int count=0，构建的时候用一个while(count<NumberOfVertices)来判断是否所有的顶点都进入了DFS Spanning Tree中，比如0被我设置为了起点，那么count++，然后DFS_Tree的节点0的那一列的第一行就被设为了count（此时count已经变为1了 ，说明节点0是第一个被加入到Spanning Tree中的），然后由于它是树根，所以第三列那个parent直接就当成-1，然后我们去edge这 个数组中寻找跟0连接的节点，将这条边纳入到DFS Spanning Tree中，然后将edge的第三列设置为1，假设通过这条边跟节点0相连的顶点为节点v，那么就在DFS_Tree这个数组中将节点v这一列的p arent设置成0，用于后面的回溯，然后将（比如节点2跟节点0相连，那么就把节点2的parent设置为0，用于后续走到头了进行回溯） ，之后就进入下一轮循环，也就是以2为点，count++....。进行完上面的步骤之后，就要开始计算每个顶点的low了，由于我刚才在DF S_Tree中已经在第三行记录了每一个节点的parent，我们找到这个parent中没有出现的点，也就是叶子。接下来要在void StronglyConnectedComponents( Graph G, void (*visit)(Vertex V) );这个函数的外面构建一个新的函数名为Find_low，需要传入我们在void StronglyConnectedComponents( Graph G, void (*visit)(Vertex V) );中构建的DFS_Tree，还有一个叶子节点（刚刚已经找到了所有的叶子节点），然后就是先计算这个叶子结点的 low，计算完成之后再计算这个叶子节点的parent的low，一直到parent=-1（也就是计算到根节点）的时候，函数终止，返回DFS_Tree （这个DFS_Tree已经更新过low了）。我们对每一个叶子节点都执行一次Find_low函数，就能够得出所有节点的low，最终就是，所有l ow相等的节点构成一个 strongly connected components in a digrap，然后把low相同的节点在同一行输出就行。

    我的错误¶

    第一¶

    low 不能 DFS 结束后统一算。而必须在DFS 回溯时动态更新。

    也就是：

    当：

    1
    u -> v

    DFS 完 v 后：

    1
    low[u] = min(low[u], low[v]);

    立刻更新。

    第二（最关键）¶

    不是low 相同就是 SCC。而是 用栈维护当前 SCC

    并在：

    1
    dfn[u] == low[u]

    时弹栈。

    void StronglyConnectedComponents( Graph G, void (*visit)(Vertex V) ){

    int n = G->NumOfVertices;
    int m = G->NumOfEdges;



    /* 构建 edge 数组 [m][3] */

    /* 第1列: 起点, 第2列: 终点, 第3列: 0=非树边, 1=树边 */

    int **edge = (int **)malloc(m * sizeof(int *));

    int edgeNumber = 0;

    for (int u = 0; u < n; u++) {

        PtrToVNode p = G->Array[u];

        while (p) {

            edge[edgeNumber] = (int *)malloc(3 * sizeof(int));

            edge[edgeNumber][0] = u;

            edge[edgeNumber][1] = p->Vert;

            edge[edgeNumber][2] = 0;

            edgeNumber++;

            p = p->Next;

        }

    }



    /* 构建 tree 数组 [3][n] */

    /* 第1行: number, 第2行: low, 第3行: parent(-1=树根) */

    int **tree = (int **)malloc(3 * sizeof(int *));

    tree[0] = (int *)calloc(n, sizeof(int));

    tree[1] = (int *)calloc(n, sizeof(int));

    tree[2] = (int *)malloc(n * sizeof(int));

    for (int i = 0; i < n; i++) tree[2][i] = -1;



    /* Tarjan 栈 */

    int *stack = (int *)malloc(n * sizeof(int));

    int top = 0;

    int *inStack = (int *)calloc(n, sizeof(int));



    /* 模拟递归栈 (迭代DFS): 每个元素 [当前顶点, 邻居迭代位置] */

    int *simStack = (int *)malloc(n * 2 * sizeof(int));

    int simTop = 0;



    int counter = 0;



    /* 外层 while: 确保所有顶点都进入生成树 */

    for (int v = 0; v < n; v++) {

        if (tree[0][v] != 0) continue;



        simStack[0] = v;

        simStack[1] = 0;

        simTop = 1;



        while (simTop > 0) {

            int u   = simStack[(simTop - 1) * 2];

            int *ni = &simStack[(simTop - 1) * 2 + 1];



            if (*ni == 0) {

                counter++;

                tree[0][u] = counter;

                tree[1][u] = counter;

                stack[top++] = u;

                inStack[u] = 1;

            }



            /* 跳过已处理的邻居 */

            PtrToVNode p = G->Array[u];

            int skip = *ni;

            while (p && skip > 0) { p = p->Next; skip--; }



            int found = 0;

            while (p && !found) {

                int w = p->Vert;

                (*ni)++;

                p = p->Next;



                if (tree[0][w] == 0) {

                    /* 树边 */

                    for (int k = 0; k < m; k++)

                        if (edge[k][2] == 0 && edge[k][0] == u && edge[k][1] == w) {

                            edge[k][2] = 1; break;

                        }

                    tree[2][w] = u;

                    simStack[simTop * 2]     = w;

                    simStack[simTop * 2 + 1] = 0;

                    simTop++;

                    found = 1;

                } else if (inStack[w]) {

                    /* back edge */

                    if (tree[0][w] < tree[1][u])

                        tree[1][u] = tree[0][w];

                }

                /* cross edge: 忽略 */

            }



            if (!found) {

                /* 回溯 */

                simTop--;

                int parent = tree[2][u];

                if (parent != -1 && tree[1][u] < tree[1][parent])

                    tree[1][parent] = tree[1][u];



                if (tree[0][u] == tree[1][u]) {

                    /* number == low → 弹栈输出一个 SCC */

                    while (1) {

                        top--;

                        int w = stack[top];

                        inStack[w] = 0;

                        visit(w);

                        if (w == u) break;

                    }

                    printf("\n");

                }

            }

        }

    }



    /* 释放内存 */

    for (int i = 0; i < m; i++) free(edge[i]);

    free(edge);

    free(tree[0]); free(tree[1]); free(tree[2]); free(tree);

                        top--;

                        int w = stack[top];

                        inStack[w] = 0;

                        visit(w);

                        if (w == u) break;

                    }

                    printf("\n");

                }

            }

        }

    }



    /* 释放内存 */

    for (int i = 0; i < m; i++) free(edge[i]);

    free(edge);

    free(tree[0]); free(tree[1]); free(tree[2]); free(tree);

    free(stack);

    free(inStack);

    free(simStack);

}
    改动说明：

    只加了两样东西，其余全是你的框架：

    为什么必须加栈？ 回溯遇到已访问的邻居 w 时：

    • inStack[w] == 1 → back edge，计入 low
    • inStack[w] == 0 → cross edge（w 已经在之前的 SCC 被弹出了），忽略

    没有 inStack，你会把 cross edge 也当成 back edge 算进 low，导致不同 SCC 被错误合并。
