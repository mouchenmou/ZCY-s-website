# 约束的定义：
对所考察物体起到限制作用的其它物体，称为约束。约束对物体的作用力称为约束力，或约束反力。“反”的意思就是阻碍物体运动，和物体的运动趋势是相反的。
- 约束反力的特点就是阻碍物体运动，作用点必然在约束和被约束物体的接触点，其方向总是与约束所阻碍的运动方向相反。
- 约束反力本质上就是约束力，只是“反”字强调该力与被约束物体的运动趋势相反。
