# 数据结构基础 FDS

FDS给我的体验就是，他把我们当成了有OI基础的选手来上课。像寻找有负权重边的Bellman-Ford算法，SPFA算法。寻找最小生成树的Prim算法和Kruskal算法这些直接一笔带过了，都是只讲算法的时间过程而不讲算法的原理，这样子我感觉真的没意义，连算法的原理都不懂，那不就是应付考试考完就忘记吗？对自己没有实质性的帮助。

像普林斯顿在Coursera上开源的Algorithm I和Algorithm II以及UCB的CS61B都能把算法的原理解释的很清楚。因此我在这一份笔记中还参杂了一些来自这两个学校的算法课的证明过程，希望对大家学习FDS起到一点帮助。
