这道题中这个地方我懵了，结构体学的实在是不好：
```C
    typedef struct Vnode{
        PtrToAdjVNode FirstEdge;
    } AdjList[MaxVertexNum];
```
我不知道为什么最后一行冒出来了一个从来没有定义过的`AdjList`

但是，这段代码实际上等同于下面两步操作：

**1. 定义结构体**

```C
    struct Vnode {
        PtrToAdjVNode FirstEdge;
    };
```

这里定义了一个名叫 `Vnode` 的结构体，里面包含一个指向邻接边节点的指针 `FirstEdge`。这代表图的顶点表（数组）里的一个单一节点。

**2. 使用 typedef 为数组定义别名** 

在 C 语言中，如果你想给一个“数组类型”起别名，语法是这样的：

`typedef 元素类型 新类型名[数组大小];`

举个例子：`typedef int NumberArray[10];` 这句代码的意思是：`NumberArray` 从此以后就是一个**数据类型**了，它代表一个包含 10 个整数的数组。当你以后写 `NumberArray myArr;` 时，就等同于写了 `int myArr[10];`。

回到题目，把刚才的 `struct Vnode` 代入这个数组定义的语法中：


```C
    typedef struct Vnode AdjList[MaxVertexNum];
```

这句话的意思是: **把 `AdjList` 定义为一个新类型，这个类型代表一个包含 `MaxVertexNum` 个 `struct Vnode` 元素的数组。

也就是说 `AdList` 的意思是一个有 $MaxVertexNum$ 个 $Vnode$ 类型的元素的数组
