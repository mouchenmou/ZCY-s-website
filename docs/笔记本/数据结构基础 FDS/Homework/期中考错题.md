## 选择题

![](附件/Pasted%20image%2020260429194206.png)
### R2-3
这道题PPT上讲过。

设计一个算法来判断表达式中的括号 `()` 是否正确匹配。最适合这个任务的数据结构是什么？ 正确答案是A

**解析：** 括号匹配的经典算法依赖于“后进先出”（LIFO - Last In First Out）的特性。

- 当我们从左到右扫描表达式时，每遇到一个左括号 `(`，我们就将其压入栈中。
    
- 每遇到一个右括号 `)`，我们就检查栈顶元素。如果栈顶是左括号，说明它们匹配，将左括号弹出栈。
    
- 如果最后表达式扫描完毕且栈为空，说明所有括号都正确匹配；如果栈不为空或者在中途遇到右括号但栈为空，则说明不匹配。
    
- 因为我们需要总是处理 **最近遇到** 的未匹配的左括号，所以栈是最适合的数据结构。

## 程序填空题

![](附件/Pasted%20image%2020260429194549.png)

- **第一空：** `child != N-1 && A[child+1] < A[child]`
- **第二空：** `Tmp > A[child]`
- **第三空：** `A[j] = Tmp`

这道题的意图是从最后一个非叶子节点开始的，我一开始以为是从第一个叶子节点开始，因为N等于7的时候的 $\frac{N-1}{2}$ 确是第一个叶子节点，但是此时直接被 $leftchild(j)<N$ 给限制住了，所以还是取不到。

至于第二空，我有一个误解，我以为每执行一次循环，Tmp都会发生一次改变，但是记住for循环里面的第一个东西，它只在进入循环的时候起一次作用！！！就像`for(int i=0; i<n; i++)`中，`i=0`只执行了一次！！！

我原本还觉得第二空可以是`A[i]>A[child]`，因为`Tmp`赋值的时候`j`跟`i`是相等的，但是！！！`A[i]`中存放的值会变化！！！就比如说第一次循环的时候，此时i跟j是相等的，当`A[child]`的值大于`Tmp`的时候，`A[j]`存放的值就会变成`A[child]`里存放的值，注意，此时`i`跟`j`相等，所以`A[i]`的值也改变了！！！
