## 1-1

> For a graph, if each vertex has an even degree or only two vertices have odd degree, we can find a **cycle** that visits every edge exactly once.

- T
- F

??? success "答案与解析"
    **答案：F**
    
    关键词是 cycle（回路，起点 = 终点）。
    
    | 条件 | 存在 |
    |------|------|
    | 所有顶点度为偶数 | **欧拉回路**（Euler circuit） |
    | 恰好 2 个顶点度为奇数 | **欧拉路径**（Euler path），但**不是回路** |
    
    题目说"only two vertices have odd degree"也能有 cycle——错误，那只够路径，不够回路。此外还需要图是连通的（忽略孤立点）。

## 1-2

> After the first run of Insertion Sort, it is possible that no element is placed in its final position.

- T
- F

??? success "答案与解析"
    **答案：T**
    
    第一趟（P=1）只处理前两个元素。即使前两个已经相对有序，它们也未必在最终该在的位置上。
    
    !!! example "反例"
        数组 `[3, 2, 1]`，升序排序：
        
        - P=1：比较 A[1]=2 与 A[0]=3，交换 → `[2, 3, 1]`
        - 此时 2 不在最终位（最终位是 A[0] 存 1），3 也不在（最终位是 A[2]），1 还没动过
        
        三个元素无一在最终位置的正确位置。
