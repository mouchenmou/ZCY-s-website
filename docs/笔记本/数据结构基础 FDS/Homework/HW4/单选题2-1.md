由于degree为3，故这棵树中孩子最多的节点有三个孩子。设没有孩子，有一个孩子，有两个孩子和有三个孩子的节点分别为$n_{0},\ n_{1}, \ n_{2}, \ n_{3}$。

则总节点数为$n_{0}+\ n_{1}+ \ n_{2}+ \ n_{3}$

由题意得$n_{3} = 2,\  n_{2}=3$

总枝条数为$n_{0}+\ n_{1}+ \ n_{2}+ \ n_{3} \ -1 = \ 3n_{3} +\ 2n_{2}+\ n_{1}$

带入$n_{3} = 2,\  n_{2}=3$ 解得 $n_{0}=8$
