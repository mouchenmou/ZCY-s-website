!!! abstract
    正定矩阵 = 对称 + 所有特征值 > 0。五种等价测试。最实用的：看主元符号。能量定义 $x^T A x > 0$ 是最基础的定义。

---

## 1. 五种等价定义

对称矩阵 A 是**正定**的（所有 λ > 0），当且仅当：

1. 所有 n 个主元 > 0
2. 所有 n 个左上角主子行列式 > 0
3. 所有 n 个特征值 > 0
4. $x^T A x > 0$ 对所有 $x \neq 0$（能量定义）
5. $A = R^T R$，R 列独立

---

## 2. $2 \times 2$ 的检验

$A = \begin{bmatrix} a & b \\ b & c \end{bmatrix}$。

**测试 1（行列式）：** $a > 0$ 且 $ac - b^2 > 0$。$ac - b^2 = \det A = \lambda_1 \lambda_2 > 0$ → λ 同号。$a > 0$ 保证了二者皆正（否则 $a < 0$ → λ 都负 → 负定）。

**测试 2（主元）：** $a > 0$ 且 $(ac-b^2)/a > 0$。第一个主元 = a，第二个主元 = $c - (b/a)b = (ac-b^2)/a$。

$A_1 = \begin{bmatrix}1&2\\2&1\end{bmatrix}$：$a>0$ 但 $\det = -3 < 0$ → 不定（λ=3,-1）。$A_2 = \begin{bmatrix}1&-2\\-2&6\end{bmatrix}$：$a=1>0, \det=2>0$ → 正定。$A_3 = \begin{bmatrix}-1&2\\2&-6\end{bmatrix}$：$a<0$ → 负定。

**测试 3（特征值）：** $\lambda_1 \lambda_2 > 0$（同号）+ $a > 0$（因 $\lambda_1 + \lambda_2 = a + c$，同号且 a>0 → 两正）。

---

## 3. 能量定义 $x^T A x > 0$（最基础）

$x^T A x = ax^2 + 2bxy + cy^2$。**配方法就是高斯消去：** $A = LDL^T$，则：

$$x^T A x = (L^T x)^T D (L^T x) = \sum d_i z_i^2$$

$d_i$ = 主元，是平方项的系数。主元全正 → 平方和恒正。

**能量定义的威力：** $A, B$ 正定 → $A+B$ 正定（$x^T(A+B)x = x^T A x + x^T B x > 0$）。用特征值/主元判断困难，能量定义直接。

---

## 4. $A^T A$ 是正定的（A 列独立时）

$$x^T(A^T A)x = (Ax)^T(Ax) = |Ax|^2 \ge 0$$

等号 ⇔ $Ax = 0$。A 列独立 → 零空间 = {0} → $x=0$ 是唯一等号情况 → **$A^T A$ 正定！** 这是最小二乘中 $A^T A$ 可逆的深层原因。

**例子：** $A = R^T R$，其中 $R$ 是矩形一阶差分矩阵：
$$\begin{bmatrix}2&-1&0\\-1&2&-1\\0&-1&2\end{bmatrix} = \begin{bmatrix}1&-1&0&0\\0&1&-1&0\end{bmatrix}\begin{bmatrix}1&0\\-1&1\\0&-1\\0&0\end{bmatrix}$$

R 三列独立 → A 正定。A 的二次型可配成 3 个平方和：$2(x_1 - \frac{1}{2}x_2)^2 + \frac{3}{2}(x_2 - \frac{2}{3}x_3)^2 + \frac{4}{3}x_3^2$——主元 2, 3/2, 4/3 是平方的系数。

**Cholesky 分解：** $A = LDL^T = (L\sqrt{D})(\sqrt{D}L^T) = R^T R$。$R = L\sqrt{D}$ 是上三角的 Cholesky 因子。`R = chol(A)` 比 LU 快一倍。

---

## 5. 半正定矩阵

正定边缘 → λ ≥ 0，有零 λ → 奇异（行列式=0）。

如 $A = \begin{bmatrix}1&2\\2&4\end{bmatrix}$：λ = 5, 0。$x^T A x = (x+2y)^2 \ge 0$。$A = \begin{bmatrix}1&0\\2&0\end{bmatrix}\begin{bmatrix}1&2\\0&0\end{bmatrix} = R^T R$，R 列不独立。

正定性检验之一不通过 → 只满足部分（λ ≥ 0 不 > 0）→ 半正定。如果哪个都不满足 → 不定。
