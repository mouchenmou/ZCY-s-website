!!! abstract
    对称矩阵 = 最重要的矩阵。两个"特权"：特征值全是实数；特征向量可选为标准正交。谱定理 $A = Q\Lambda Q^T$ = 正交投影的加权和。证明用共轭复数技巧。

---

## 1. 谱定理：$A = Q\Lambda Q^T$

对称矩阵 $A = A^T$。对角化 $A = S\Lambda S^{-1}$。取转置：$A^T = (S^{-1})^T \Lambda S^T = A = S\Lambda S^{-1}$。如果 $S^{-1} = S^T$ 即 $S$ 正交 → Q，完美。对称矩阵恰好保证了这一点。

**定理（谱定理 / 主轴定理）：每实对称矩阵 A 有分解 $A = Q\Lambda Q^T$：**
- Λ 对角元全是**实数**特征值
- Q 的列是**标准正交**特征向量（$Q^T Q = I$，$Q^{-1} = Q^T$）

证明分两步：(1) λ 全是实数；(2) 特征向量可正交化。

---

## 2. 性质一：特征值全是实数——复数取共轭证明

设 $Ax = \lambda x$。λ, x 可能是复数。取共轭：$A\bar{x} = \bar{\lambda}\bar{x}$（A 是实矩阵，$\bar{A}=A$）。转置：$\bar{x}^T A^T = \bar{x}^T \bar{\lambda}$。因 $A^T = A$（对称！）：$\bar{x}^T A = \bar{x}^T \bar{\lambda}$。

**关键——用两种方式算 $\bar{x}^T A x$：**
- 从 $Ax = \lambda x$：$\bar{x}^T A x = \bar{x}^T \lambda x = \lambda \bar{x}^T x$
- 从 $\bar{x}^T A = \bar{x}^T \bar{\lambda}$：$\bar{x}^T A x = \bar{x}^T \bar{\lambda} x = \bar{\lambda} \bar{x}^T x$

两式相等：$\lambda \bar{x}^T x = \bar{\lambda} \bar{x}^T x$。$\bar{x}^T x = |x_1|^2 + \cdots + |x_n|^2 > 0$（x ≠ 0 的特征向量）。所以 $\lambda = \bar{\lambda}$ → **虚部为 0，λ 是实数。** QED。

**推导关键：** 利用对称性使 $A^T = A$，对原式先取共轭再转置后，二者产生同一 $\bar{x}^T A x$，逼出 $\lambda = \bar{\lambda}$。如果 A 不是对称的，此论证就行不通。

---

## 3. 性质二：不同 λ 对应的特征向量正交

设 $Ax = \lambda_1 x, Ay = \lambda_2 y, \lambda_1 \neq \lambda_2$。

$$(\lambda_1 x)^T y = (Ax)^T y = x^T A^T y = x^T A y = x^T(\lambda_2 y) \tag{3}$$

关键一步：$(Ax)^T = x^T A^T = x^T A$（因对称）。所以 $x^T \lambda_1 y = x^T \lambda_2 y$ → $(\lambda_1 - \lambda_2) x^T y = 0$ → $x^T y = 0$ ✓。

如果 λ 重复 → 对应一个多维特征空间，可在内部选正交基。**总可选出 n 个互相正交的特征向量** → 归一化得 Q。

**2×2 对称矩阵的特征向量特殊形式：**

$$A = \begin{bmatrix} a & b \\ b & c \end{bmatrix}: x_1 = \begin{bmatrix}b\\\lambda_1-a\end{bmatrix}, x_2 = \begin{bmatrix}\lambda_2-c\\b\end{bmatrix} \tag{4}$$

验证：$x_1^T x_2 = b(\lambda_2-c) + (\lambda_1-a)b = b(\lambda_1+\lambda_2-a-c) = 0$（因 $\lambda_1+\lambda_2 = \text{trace} = a+c$）。

---

## 4. $A = Q\Lambda Q^T$ 的秩一分解

$$A = Q\Lambda Q^T = \lambda_1 q_1 q_1^T + \cdots + \lambda_n q_n q_n^T \tag{6}$$

每个 $\lambda_i q_i q_i^T$ 是秩一的投影矩阵（$q_i q_i^T$ 投影到 $q_i$ 方向，因 $q_i$ 是单位向量）。**对称矩阵 = 互相垂直的投影矩阵的加权和。** 这是谱分解的几何本质。

---

## 5. 实数矩阵的复数特征值

非对称矩阵可能产生复数 λ。旋转矩阵 $Q = \begin{bmatrix}\cos\theta&-\sin\theta\\\sin\theta&\cos\theta\end{bmatrix}$：λ = $e^{\pm i\theta}$。**实数矩阵的复数 λ 以共轭对出现：$Ax = \lambda x$ → $A\bar{x} = \bar{\lambda}\bar{x}$。**

正交矩阵：$|\lambda| = 1$。反对称矩阵：$\lambda$ 是纯虚数。**对称矩阵像实数，反对称矩阵像虚数，正交矩阵像 $|\lambda| = 1$ 的复数。** 这是定理，不是类比。

---

## 6. 特征值 vs 主元

对对称矩阵：**正主元数 = 正特征值数。** 乘积关系：无行交换时主元乘积 = 行列式 = 特征值乘积。但单个主元和特征值没有简单对应。
