!!! abstract
    特征值在动态问题中发挥最重要的作用。$du/dt = Au$ 的解和 $u_{k+1}=Au_k$ 完全一样的三步法——只是 $\lambda^k$ 变成 $e^{\lambda t}$。稳定性由 $\lambda$ 实部决定。

---

## 1. 把微分方程转换为线性代数

标量方程 $du/dt = \lambda u$ → $u(t) = u(0) e^{\lambda t}$（当 $t=0$，$e^{\lambda t}=1$，所以 $C = u(0)$）。

向量方程 $d\vec{u}/dt = A\vec{u}$ 同样求解。$A$ 是常系数矩阵，$\vec{u}$ 的每个分量对 $t$ 求导。

**关键：** 设 $u(t) = e^{\lambda t} x$（$x$ 是不随时间变化的**固定向量**）。代入 $du/dt = Au$：
- $du/dt = \lambda e^{\lambda t} x$（把 $\lambda$ 提出来）
- $Au = A(e^{\lambda t} x) = e^{\lambda t} Ax$

两者相等需要 $Ax = \lambda x$。**所以 $x$ 必须是特征向量，$\lambda$ 必须是特征值！**

**为什么 $u$ 的分量必须是 $e^{\lambda t} x$ 的形式？** 因为 $du/dt = Au$ 意味着导数 = 常数 × 原向量——这正是指数函数的性质。求导提下 $\lambda$；$A$ 乘特征向量 = $\lambda$ 乘同一向量。二者恰好吻合。

**纯指数解：** $u(t) = e^{\lambda t} x$。$\lambda > 0$ → 增长；$\lambda < 0$ → 衰减。复数 $\lambda$ → 实部决定增长/衰减，虚部产生震荡 $e^{i\omega t}$。

---

## 2. 完整解的三步法（和差分方程完全一样！）

1. 把 $u(0)$ 写成特征向量的组合：$u(0) = c_1 x_1 + \cdots + c_n x_n$
2. 每个特征向量乘 $e^{\lambda_i t}$：$c_i e^{\lambda_i t} x_i$
3. 加起来：$u(t) = c_1 e^{\lambda_1 t} x_1 + \cdots + c_n e^{\lambda_n t} x_n$

!!! example "例 1"
    $du/dt = \begin{bmatrix}0&1\\1&0\end{bmatrix} u, u(0) = \begin{bmatrix}4\\2\end{bmatrix}$。

    $A$ 的特征值 $\lambda=1,-1$，特征向量 $(1,1), (1,-1)$。纯指数解：$u_1 = e^t(1,1)^T, u_2 = e^{-t}(1,-1)^T$。

    通解：$u = C e^t(1,1)^T + D e^{-t}(1,-1)^T$。$t=0$：$C(1,1) + D(1,-1) = (4,2)$ → $C=3, D=1$。

    $$u(t) = 3e^t\begin{bmatrix}1\\1\end{bmatrix} + e^{-t}\begin{bmatrix}1\\-1\end{bmatrix}$$

    **分析：** 原方程是两个耦合的标量方程 $dy/dt = z, dz/dt = y$。特征向量将它们去耦：
    $\frac{d}{dt}(y+z) = y+z$（增长，$\lambda=1$），$\frac{d}{dt}(y-z) = -(y-z)$（衰减，$\lambda=-1$）。

**例 2：** $A = \begin{bmatrix}1&1&1\\0&2&1\\0&0&3\end{bmatrix}, u(0) = \begin{bmatrix}9\\7\\4\end{bmatrix}$。特征值 $1,2,3$（三角），$x_1=(1,0,0), x_2=(1,1,0), x_3=(1,1,1)$。

$u(0) = 2x_1 + 3x_2 + 4x_3$。$u(t) = 2e^t x_1 + 3e^{2t} x_2 + 4e^{3t} x_3$。

---

## 3. 稳定性

| 特征值 | 行为 |
|--------|------|
| 所有 $\operatorname{Re}(\lambda) < 0$ | $u(t) \to 0$（稳定） |
| 某些 $=0$，其余 $<0$ | 收敛到稳态 |
| 存在 $\operatorname{Re}(\lambda) > 0$ | 发散 |
| 复数 $\lambda = a \pm bi$ | $e^{at}$ 乘震荡 |

对于 $2 \times 2$ 矩阵：迹 < 0 且行列式 > 0 → 特征值实部 < 0 → 稳定。

---

## 4. 二阶方程化为一阶系统

$my'' + by' + ky = 0$。设 $u = \begin{bmatrix}y'\\y\end{bmatrix}$：

$$u' = \begin{bmatrix} y'' \\ y' \end{bmatrix} = \begin{bmatrix} -b/m & -k/m \\ 1 & 0 \end{bmatrix} \begin{bmatrix} y' \\ y \end{bmatrix} = Au$$

第二行是恒等式 $y' = y'$。$A$ 的特征值正是 $m\lambda^2 + b\lambda + k = 0$ 的两个根——和把 $y = e^{\lambda t}$ 代入原微分方程得到的特征方程**一模一样**。矩阵 $A$ 将二阶问题封装为一阶系统。

对任意 $k$ 阶方程：$A$ 的第一行是系数，对角线下方全是 1，其余 0。$A$ 是 $k \times k$ 的。
