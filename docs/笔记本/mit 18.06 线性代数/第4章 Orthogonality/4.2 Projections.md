!!! abstract
    当 $Ax=b$ 无解——把 $b$ 投影到 $A$ 的列空间。三个步骤：$\hat{x} = (A^T A)^{-1} A^T b$，$p = A\hat{x}$，$P = A(A^T A)^{-1} A^T$。$P^2 = P$，$P^T = P$。

---

## 1. 为什么需要投影

$Ax=b$ 可能无解——b 不在列空间。但 $Ax$ 永远在列空间里。做法：把 b 投影到列空间得 $p$，解 $A\hat{x} = p$。误差 $e = b-p$ 和列空间垂直。

**两个问题开始：** b=(2,3,4) 投影到 z 轴和 xy 平面分别是什么？什么样的矩阵能产生投影？

![](附件/image-20210121092259773.png)

- 投影到 z 轴：$p_1 = (0,0,4)$，$P_1 = \begin{bmatrix}0&0&0\\0&0&0\\0&0&1\end{bmatrix}$（秩一）
- 投影到 xy 平面：$p_2 = (2,3,0)$，$P_2 = \begin{bmatrix}1&0&0\\0&1&0\\0&0&0\end{bmatrix}$（秩二）

$p_1+p_2 = b$，$P_1+P_2 = I$。**$P$ 投影到一个子空间，$I-P$ 投影到正交补。**

---

## 2. 投影到一条直线——三步

把 b 投影到方向 a 的直线上。设 $p = \hat{x}a$。

**步骤 1——$\hat{x}$：** 误差 $e = b - \hat{x}a$ 和 a 垂直：
$$a^T(b - \hat{x}a) = 0 \Rightarrow a^Tb - \hat{x}a^Ta = 0 \Rightarrow \hat{x} = \frac{a^T b}{a^T a} \tag{2}$$

**步骤 2——$p$：** $p = \hat{x}a = \frac{a^T b}{a^T a} a$

**步骤 3——$P$：** $p = \frac{aa^T}{a^T a} b \Rightarrow P = \frac{aa^T}{a^T a}$（$m \times m$ 秩一矩阵）

![](附件/image-20210121095127115.png)

$P$ 的列空间就是 a 张成的直线。$P^2 = P$——投影再投影不变。

**特殊情况：** b=a → $\hat{x}=1$, p=a；b ⟂ a → $a^T b=0$, p=0。

**例 1：** b=(1,1,1) 投影到 a=(1,2,2)。$a^T b=5,a^T a=9$ → $\hat{x}=5/9$，$p=(5/9, 10/9, 10/9)$，$e=(4/9, -1/9, -1/9)$，$e \cdot a=0$ ✓。$P = \frac{1}{9}\begin{bmatrix}1&2&2\\2&4&4\\2&4&4\end{bmatrix}$。

---

## 3. 投影到子空间——正规方程

A 的列 $a_1,\ldots,a_n$ 是子空间的基（独立）。找 $p = A\hat{x}$ 最接近 b。误差 $e = b - A\hat{x}$ 和整个列空间垂直 → 和每列都垂直：

$$a_1^T(b-A\hat{x}) = 0,\; \ldots,\; a_n^T(b-A\hat{x}) = 0 \tag{4}$$

这 n 个方程合起来：$A^T(b - A\hat{x}) = 0$ → $\boxed{A^T A \hat{x} = A^T b} \tag{5}$。这就是**正规方程**。

解：$\hat{x} = (A^T A)^{-1} A^T b$，$p = A\hat{x} = A(A^T A)^{-1} A^T b$，$\boxed{P = A(A^T A)^{-1} A^T} \tag{7}$。

---

## 4. $A^T A$ 的可逆性

**定理：$A^T A$ 可逆 ⇔ A 的列独立。** 证明：证明 $N(A^T A) = N(A)$。

(1) $Ax=0$ → $A^T A x = 0$。
(2) $A^T A x = 0$ → $x^T A^T A x = (Ax)^T (Ax) = |Ax|^2 = 0$ → $Ax = 0$。不能左乘 $(A^T)^{-1}$（不存在），而是左乘 $x^T$ 用向量长度。

$A^T A$ 是 $n \times n$ 方阵，对称。A 列独立 → $N(A^T A) = \{0\}$ → 可逆。

**例子：** 依赖列：$\begin{bmatrix}1&1&0\\2&2&0\end{bmatrix}\begin{bmatrix}1&2\\1&2\\0&0\end{bmatrix} = \begin{bmatrix}2&4\\4&8\end{bmatrix}$（奇异）。独立列：$\begin{bmatrix}1&1&0\\2&2&1\end{bmatrix}\begin{bmatrix}1&2\\1&2\\0&1\end{bmatrix} = \begin{bmatrix}2&4\\4&9\end{bmatrix}$（可逆）。

---

## 5. 投影矩阵的性质

1. **$P^T = P$**（对称）。$P^T = (A(A^T A)^{-1} A^T)^T = A((A^T A)^{-1})^T A^T = A(A^T A)^{-1} A^T = P$（因 $(A^T A)^{-1}$ 对称）
2. **$P^2 = P$**（幂等）。$P^2 = A(A^T A)^{-1} A^T \cdot A(A^T A)^{-1} A^T = P$
3. **除非 $P=I$，投影矩阵是奇异的。** $\text{rank}(P) \le \text{rank}(A) \le n < m$（当 $m > n$）。$m=n$ 时 A 可逆 → $P = I$
4. $I-P$ 投影到左零空间（正交补）
5. 误差 $e = b-p$ 既在 $P$ 的零空间（$Pe=0$）又在 $P$ 的左零空间（$P^T e = Pe = 0$）——对称矩阵两者相同

**完整算例：** A = $\begin{bmatrix}1&0\\1&1\\1&2\end{bmatrix}$, b = $(6,0,0)$。

$A^T A = \begin{bmatrix}3&3\\3&5\end{bmatrix}$，$A^T b = \begin{bmatrix}6\\0\end{bmatrix}$。解 $\hat{x} = (5,-3)$。$p = (5,2,-1)$，$e = (1,-2,1)$。$e \perp (1,1,1)$ 且 $e \perp (0,1,2)$ ✓。$P = \frac{1}{6}\begin{bmatrix}5&2&-1\\2&2&2\\-1&2&5\end{bmatrix}$。

**例：** 心跳三次测量 x=70, x=80, x=120。$A=[1,1,1]^T$, b=(70,80,120)。正规方程：$3\hat{x}=270$ → $\hat{x}=90$。这就是均值。
