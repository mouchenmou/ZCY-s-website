!!! abstract
    标准正交基使一切变简单：$Q^T Q = I$，投影和最小二乘无需求逆。Gram-Schmidt 把独立向量变成正交向量。$A=QR$ 是正交版的 $LU$ 分解。

---

## 1. 标准正交基和矩阵 Q

向量 $q_1,\ldots,q_n$ 是**标准正交**的：$q_i^T q_j = 0\;(i \neq j),\; q_i^T q_j = 1\;(i=j)$。

把这些向量作为列的矩阵记作 Q。**不管 Q 是不是方阵，$Q^T Q = I$！**

$Q^T Q = \begin{bmatrix}q_1^T\\\vdots\\q_n^T\end{bmatrix}\begin{bmatrix}q_1&\cdots&q_n\end{bmatrix} = \begin{bmatrix}1&0&\cdots&0\\0&1&\cdots&0\\\vdots&\vdots&\ddots&\vdots\\0&0&\cdots&1\end{bmatrix} = I \tag{1}$

若 Q 是方阵：$Q^T = Q^{-1}$——转置 = 逆！这时 Q 称为**正交矩阵**。

如果列只正交没归一化，$Q^T Q$ 还是对角矩阵（只是对角元 ≠ 1）。

---

## 2. 三个经典正交矩阵

**旋转：** $Q = \begin{bmatrix}\cos\theta&-\sin\theta\\\sin\theta&\cos\theta\end{bmatrix}$。$Q^T = Q^{-1}$ 旋转回去。

**置换：** $P = \begin{bmatrix}0&1&0\\0&0&1\\1&0&0\end{bmatrix}$。列是 $I$ 的排列。所有置换矩阵都是正交矩阵。

**反射：** 对单位向量 $u$，$Q = I - 2uu^T$。$Q^T = Q$，$Q^2 = I$（两次反射回到原位）。证明：$Q^T Q = (I-2uu^T)^2 = I - 4uu^T + 4u(u^T u)u^T = I - 4uu^T + 4uu^T = I$。

## 3. Q 不改变长度和角度

$|Qx|^2 = (Qx)^T(Qx) = x^T Q^T Q x = x^T x = |x|^2$。$(Qx)^T(Qy) = x^T Q^T Q y = x^T y$。**长度、内积、角度全部不变。**

---

## 4. 用正交基投影——无需求逆

子空间的基是 Q 的列时：$\hat{x} = Q^T b$（$Q^T Q = I$），$p = Q\hat{x} = Q Q^T b$，$P = Q Q^T$。**不需要任何逆！**

$\hat{x}_i = q_i^T b$——只是点乘。$p = q_1(q_1^T b) + \cdots + q_n(q_n^T b)$ = 各标准正交方向的投影之和。

**若 Q 是方阵：** $P = QQ^T = I$。$b = \sum q_i(q_i^T b)$——傅里叶级数的矩阵本质。

**例 4：** $Q = \frac{1}{3}\begin{bmatrix}-1&2&2\\2&-1&2\\2&2&-1\end{bmatrix}$，$b=(0,0,1)$。$\frac{2}{3}q_1 + \frac{2}{3}q_2 - \frac{1}{3}q_3 = b$——完美重建！

---

## 5. Gram-Schmidt 正交化

给定独立向量 $a, b, c$，得到正交 $A, B, C$：

- $A = a$
- $B = b - \frac{A^T b}{A^T A} A$（减去在 A 上的投影）
- $C = c - \frac{A^T c}{A^T A} A - \frac{B^T c}{B^T B} B$（减去在 A 和 B 上的投影）

归一化：$q_1 = A/|A|, q_2 = B/|B|, q_3 = C/|C|$。

---

## 6. $A = QR$ 分解

$$A = QR$$

Q 的列标准正交，R 是上三角。$R_{ij} = q_i^T a_j$（$i \le j$）。$a_1$ 只在 $q_1$ 有分量，$a_2$ 只在 $q_1,q_2$ 有分量…所以 R 对角线以下 = 0。

$A=QR$ 和 $A=LU$ 并列——$LU$ 来自高斯消去，$QR$ 来自 Gram-Schmidt。用 $QR$ 解最小二乘：$R\hat{x} = Q^T b$，R 上三角，回代即可，比 $(A^T A)^{-1}$ 更稳定。
