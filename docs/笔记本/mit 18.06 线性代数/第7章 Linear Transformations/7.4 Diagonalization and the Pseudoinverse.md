!!! abstract
    选对基，矩阵变成对角的。一组基（特征向量）→ Λ；两组基（奇异向量）→ Σ。伪逆 $A^+ = V\Sigma^+ U^T$ 把矩形阵"逆"回去。极坐标分解 = 正交 × 半正定。

---

## 1. 两组基的对角化

**1 组基（输入=输出，特征向量）：** $S^{-1}AS = \Lambda$。以 $x_i$ 为基，$T(x_i) = \lambda_i x_i$，在新基下坐标 = $(\ldots, \lambda_i, \ldots)^T$ → 矩阵变成对角的 Λ。需要方阵、可对角化。**特征向量基让变换变得无比简单。**

**2 组基（输入≠输出，奇异向量）：** $U^{-1}AV = \Sigma$。适用于**任何**矩阵。V = $A^T A$ 的特征向量（输入基，标准正交），U = $AA^T$ 的特征向量（输出基，标准正交）。$U^{-1}=U^T, V^{-1}=V^T$。

**变基视角：** $\Sigma = U^{-1}_{\text{standard to u}} \cdot A_{\text{standard}} \cdot V_{\text{v to standard}}$

1. V 把输入从 v 基变到标准基
2. A 在标准基下变换
3. $U^{-1}$ 把输出变回 u 基

---

## 2. SVD 的对角化

$A v_j = \sigma_j u_j$（$j \le r$），$A v_j = 0$（$j > r$）。$v_1,\ldots,v_r$ 是行空间的标准正交基，$u_1,\ldots,u_r$ 是列空间的标准正交基。$v_{r+1},\ldots,v_n$ 是零空间基，$u_{r+1},\ldots,u_m$ 是左零空间基。

$A V = U \Sigma$ → $A = U \Sigma V^T$。

---

## 3. 伪逆 $A^+$

矩形矩阵没有双边逆。但利用 SVD 可定义伪逆：$A^+ = V \Sigma^+ U^T$。$\Sigma^+$ 是将 Σ 的非零对角元 $\sigma_i$ 变成 $1/\sigma_i$，再转置。

**$A^+$ 的作用：** 把向量从 $R^m$ 送回 $R^n$。列空间的向量 → 映回行空间中对应的**唯一**向量；左零空间的向量 → 映为 0。

**求导/积分：** 导数矩阵 A（3×4）的伪逆 = 积分矩阵 $A_{\text{int}}$（4×3）。$A A_{\text{int}} = I$，$A_{\text{int}} A = \text{diag}(0,1,1,1)$。

**线性变换的语言：** $A^+$ 逆转了 A 的映射。$A$ 把行空间唯一地映到列空间；$A^+$ 把列空间唯一地映回行空间。零空间和左零空间被消灭。

---

## 4. 极坐标分解

任何实数方阵 = 正交 × 半正定：$A = QH$。

**伪逆的关键性质：**
- $A^+ u_i = v_i / \sigma_i$（$i \le r$），$A^+ u_i = 0$（$i > r$）。$A^+$ 把列空间的 $u_i$ 送回行空间，消灭左零空间的向量。
- $\Sigma^+ \Sigma = \begin{bmatrix} I_r & 0 \\ 0 & 0 \end{bmatrix}$——尽可能接近单位矩阵（r 个 1，其余 0）。
- $A A^+$ = 投影到列空间的投影矩阵；$A^+ A$ = 投影到行空间的投影矩阵。

![](附件/image-20210705164146530.png)

A 的作用：将行空间向量 $x_r$ → 列空间，将零空间 $x_n$ → 0。任何 $R^n$ 中向量 x 都是行空间+零空间分量之和。A 把 $R^n$ 切换到 $R^m$。

伪逆 $A^+$ 逆转这个映射：列空间向量 p 可由 $A x^+ = p$ 得到（$x^+$ 在行空间），而 $A^+ p = x^+$。对于任意 b = p + e（p 在列空间，e 在左零空间），$A^+ b = x^+$——因为 $A^+$ 消去左零空间的误差 e。

**例 4：秩 1 矩阵。** $A = \begin{bmatrix}2&2\\1&1\end{bmatrix}$，唯一的 σ = $\sqrt{10}$。$A^+ = \frac{1}{10}\begin{bmatrix}2&1\\2&1\end{bmatrix}$。$A^+$ 秩也是 1，列空间 = A 的行空间。$A(1,1)^T = (4,2)^T$；$A^+(4,2)^T = (1,1)^T$。

一般秩 1 矩阵：$A = \sigma u v^T$（u, v 单位向量）→ $A^+ = v u^T / \sigma$。$A A^+ = u u^T$（投影到 u 线），$A^+ A = v v^T$（投影到 v 线）。

---

## 5. 总结

| 分解 | 本质 |
|------|------|
| $A=LU$ | 消去——输出是三角，输入基不动 |
| $A=QR$ | 输出基变 Q（正交），输入基 = 标准基 |
| $S^{-1}AS=\Lambda$ | 输入=输出基 = 特征向量 → 对角 |
| $U^{-1}AV=\Sigma$ | 两组独立正交基 → 对角（SVD） |
| $A^+=V\Sigma^+U^T$ | 逆转 SVD 映射 |
| $A=QH$ | 正交旋转 × 对称拉伸（极坐标） |
