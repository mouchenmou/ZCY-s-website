!!! abstract
    同一空间下变基：$x_{\text{old}} = Wc_{\text{new}}$。同一个变换在不同基下的矩阵相似：$B = M^{-1}AM$。选特征向量为基 → $\Lambda = S^{-1}AS$。SVD = 两组基的对角化。

---

## 1. 变基矩阵

设旧基（标准基），新基 $w_1,\ldots,w_n$。W 的列 = 新基在旧基下的坐标。

$$x_{\text{old}} = W c_{\text{new}}, \quad c_{\text{new}} = W^{-1} x_{\text{old}}$$

W 把新坐标变旧坐标；$W^{-1}$ 反过来。W 和 $W^{-1}$ 代表恒等变换——向量本身没变，坐标变了。

---

## 2. 同一变换在不同基下的矩阵：相似变换

变换 T 在两个基下对应 A 和 B。$B = M^{-1}AM$——A 和 B **相似**。

**三步理解 $B = M^{-1}AM$：**
1. M 把新基坐标变成旧基坐标
2. A 在旧基下做变换
3. $M^{-1}$ 把结果变回新基坐标

这是变换的复合：$ITI$ = 先变基 → 变换 → 再变回来。

**对于同一个空间下的变基（输入=输出），$B = M^{-1}AM$ 就是矩阵的相似变换。** 当 M = S（特征向量矩阵）→ $\Lambda = S^{-1}AS$。

**例（投影到 45° 线，三组基对比）：**

投影到 $y=-x$。A 特征向量 $x_1=(1,-1)$（在直线上，λ=1），$x_2=(1,1)$（垂直，λ=0）。

- **标准基：** $A = \begin{bmatrix}0.5&-0.5\\-0.5&0.5\end{bmatrix}$
- **特征向量基：** $\Lambda = \begin{bmatrix}1&0\\0&0\end{bmatrix}$（对角的！）
- **另一组基 $w_1=(2,0), w_2=(1,1)$：** $T(w_1)=(1,-1)=w_1-w_2$ → 第 1 列 $(1,-1)^T$；$T(w_2)=0$ → 第 2 列 $(0,0)^T$。$B = \begin{bmatrix}1&0\\-1&0\end{bmatrix}$

验证 $B = W^{-1}AW$：$W=\begin{bmatrix}2&1\\0&1\end{bmatrix}, W^{-1}=\begin{bmatrix}0.5&-0.5\\0&1\end{bmatrix}, W^{-1}AW = B$ ✓。A、B、Λ 三者相似——都有特征值 1,0 且 $P^2=P$。

---

## 3. 几何重述

- **特征向量基的几何含义：** 特征向量是变换的"不变方向"——A 只缩放不旋转。以不变方向为坐标轴，A 变成纯粹的对角放大矩阵 Λ。
- **相似变换的几何含义：** 同一个线性变换，不同坐标系下有不同的矩阵表达式。相似变换 $B = M^{-1}AM$ 恰好对应坐标系切换。
- **SVD 的变基视角：** $\Sigma = U^{-1} A V$——输入和输出空间使用**不同的基**。输入基 V（$A^T A$ 的特征向量），输出基 U（$AA^T$ 的特征向量）。两组标准正交基，把任意矩阵对角化为 Σ。

**所有伟大的分解都是变基：**
| 分解 | 本质 |
|------|------|
| $S^{-1}AS = \Lambda$ | 输入=输出基 = 特征向量 |
| $U^{-1}AV = \Sigma$ | 输入基 v，输出基 u（SVD） |

## 4. 变基的坐标视角

设 $B = \{v_1,v_2\}$ 是 $R^2$ 的一组基（$v_1 = (2,1), v_2 = (1,2)$）。$a = 3v_1 + 2v_2 = (8,7)$。**a 在标准基下的坐标是 $(8,7)$，在基 B 下的坐标是 $(3,2)$。**

![](附件/1617279583103.png)

**不同的基下，同一个向量的坐标不一样。** 标准基下的坐标直接写为 $\vec{a}$，基 B 下写为 $[\vec{a}]_B$。

---

## 5. 变换矩阵是绑定在基上的

设变换 $T: R^n \to R^n$ 可用矩阵 A 表示：$T(\vec{x}) = A\vec{x}$。

![](附件/image-20210402111047162.png)

但这里隐含了：$\vec{x}$ 是基于标准基的。更严谨的说法是：**A 是 T 关于标准基的变换矩阵。同一个变换，在不同基下的变换矩阵不一样！**

---

## 6. 困难变换 → 先变基

对于关于直线 $L: y = 2x$ 的反射变换，直接通过几何确定 $(1,0)$ 和 $(0,1)$ 变换后的位置很困难。

![](附件/image-20210402140820707.png)

**但可以变基！** 把 $v_1$（在 L 上）和 $v_2$（垂直于 L）作为新基。在新基下，反射只是把 $v_2$ 变号：$D = \begin{bmatrix}1&0\\0&-1\end{bmatrix}$。然后用 $A = W D W^{-1}$ 变回标准基——这就是 $B = M^{-1}AM$ 的威力。
| $B = W^{-1}AW$ | 同一变换，不同基 → 相似 |
