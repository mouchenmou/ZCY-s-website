!!! abstract
    线性变换是学习线代的另一个开始点。即使不知道什么是矩阵，也能理解线性变换。物理学家不关心坐标系和坐标值，只关心如何描述变换。每个线性变换都对应一个矩阵，矩阵的背后正是线性变换的概念。

---

## 1. 什么是线性变换

不通过任何矩阵，也可以通过线性变换描述一个投影。如，把 $R^2$ 平面内的一个向量变成 $R^2$ 平面内另一个向量（$R^2 \to R^2$），这种关系通常称为映射（mapping）。假设平面上有一条直线，把平面内的所有向量 v 投影到这条直线上，线性变换生成另一个向量 $T(v)$，$T(v)$ 就是投影到直线上的向量。T 就像一个函数，对某输入进行变换得到一个输出。这里其实没有坐标——也可以把坐标轴去掉。

当矩阵 A 乘以向量 v，就变换 v 为另外一个向量 Av。输入是 v，输出是 $T(v) = Av$。变换 T 的概念和函数是一样的。更重要的是：**一次性看清楚所有 v 的变换。当所有 v 都乘 A 时，我们变换的是所有 v 组成的空间 V。**

A 把 v 变换为 Av，把 w 变换为 Aw。u = v + w 会发生什么？**Au 就是 Av + Aw。** 矩阵乘法 $T(v) = Av$ 是线性变换。

!!! definition "线性变换的定义"
    一个变换 T 对 V 中的每个输入向量 v 都给出一个输出 T(v)。如果对所有 v, w 都满足下面两个条件，则 T 是**线性的（linear）**：
    
    1. $T(v+w) = T(v) + T(w)$
    2. $T(cv) = cT(v)$（对所有 c）
    
    这其实就是向量的加法和数乘。线性变换应保证这两种运算的不变性。合起来：
    
    $$T(cv+dw) = cT(v) + dT(w) \tag{M1}$$
    
    任何 v,w 线性组合的线性变换，肯定会产生对 T(v), T(w) 的同样的线性组合。矩阵乘法是线性的：$A(cv+dw) = cAv + dAw$。

**如果输入 v = 0，输出 T(v) 必须是 0。零向量通过线性变换一定是零向量——可以通过 T(0) 来判断是不是线性变换。**

## 2. 非线性的变换

**例 V2（平移）：** 把 $u_0$ 加到每一个变换向量上，$T(v) = v + u_0, T(w) = w + u_0$。这不是线性的！

$$T(v+w) = v + w + u_0 \quad\text{不等于}\quad T(v) + T(w) = v + u_0 + w + u_0$$

例外是 $u_0 = 0$，此时 $T(v) = v$ 是恒等变换（identity transformation），明显是线性的。线性-平移的变换 $T(v) = Av + u_0$ 称为**仿射（affine）**。虽然 T 不再是线性的，但直线还是直线。计算机图形学的仿射变换将在第 8 章学习——因为必须能够移动图片。

**例 1（点乘）：** $a = (1,3,4)$，$T(v) = a \cdot v = v_1 + 3v_2 + 4v_3$。这是线性的。V = $R^3$，W = $R^1$。这其实就是 $A = [1,3,4]$ 乘以 v。

如果输出包含乘积或长度：$v_1^2$ 或 $v_1 v_2$ 或 $|v|$，则 T 不是线性的。

**例 2（长度）：** $T(v) = |v|$ 不是线性的。虽然向量翻倍长度也翻倍，但乘 -2 长度还是翻倍而不是乘 -2，错了。$|v+w| \le |v|+|w|$，$|-v| \ne -|v|$——两条都不成立。

**例 3——重要（旋转）：** T 把任何向量旋转 30 度。定义域是 xy 平面，值域也是 xy 平面。没有矩阵时，这样描述 T：旋转 30 度。旋转是线性的——旋转两个向量后加在一起，和 T(v) + T(w) 与 T(v+w) 是一样的。整个输入平面都旋转了。

**例 V3（旋转 45 度）：** 平面内一个向量映射到平面内另一个向量，逆时针旋转 45 度。

![](附件/image-20210701172345101.png)

如果 v 翻倍，旋转后的向量同样翻倍。对于 v+w，无论先旋转后加还是先加后旋转，结果一样——所以是线性变换。

## 3. 直线还是直线，三角形还是三角形

![](附件/image-20210329165335473.png)

Fig7.1 最左边展示了输入空间 v 到 w 的直线，也展示了输出空间 T(v) 到 T(w) 的直线。线性化告诉我们：**输入线上的每个点，都到了输出线的点上。而且等距的点在输出空间还是等距的点。** 输入线上的中心点 $u = \frac12 v + \frac12 w$，变换到输出线上的中心点：$T(u) = \frac12 T(v) + \frac12 T(w)$。

Fig7.1 右边增加了一维：3 个顶点 $v_1, v_2, v_3$。输入的三角形，输出还是一个三角形。等距的点还是等距的。中心点 $u = \frac13(v_1+v_2+v_3)$，输出的中心点 $T(u) = \frac13(T(v_1) + T(v_2) + T(v_3))$。

**线性化的规则可扩展到 3 个或 n 个向量的组合：**

$$u = c_1 v_1 + \cdots + c_n v_n \quad\text{变换到}\quad T(u) = c_1 T(v_1) + \cdots + c_n T(v_n) \tag{1}$$

## 4. 变换的语言：值域和内核

没有矩阵时，不能讨论列空间，但概念可以继续使用。变换用"值域"和"内核"来表示：

- **T 的值域（Range）：** 所有输出 T(v) 的集合（对应列空间 Av）
- **T 的内核（Kernel）：** 所有使 T(v) = 0 的输入的集合（对应零空间）

值域在输出空间 W，内核在输入空间 V。当 T 可用矩阵乘法 $T(v) = Av$ 表示时，值域 = C(A)，内核 = N(A)。

**例 4（投影到 xy 平面）：** $T(x,y,z) = (x,y,0)$。值域是这个平面，包含所有 T(v)。内核是 z 轴（投影为 0）。这个投影是线性的。

**例 5（投影到 z=1）：** $T(x,y,z) = (x,y,1)$ 不是线性的——甚至不能把 v=0 投影到 T(v)=0。

3×3 矩阵 A 乘任何三维向量，$T(v) = Av$ 是线性的：$T(v+w) = A(v+w) = Av + Aw = T(v) + T(w)$。

**例 6：** T 可用可逆矩阵 A 表示。内核是零向量，值域 W 等于定义域 V。逆变换 $T^{-1}$ 把任何 T(v) 变回 v：$T^{-1}(T(v)) = v$，符合 $A^{-1}(Av) = v$。

**重要问题：** 所有从 $V=R^n$ 到 $W=R^m$ 的线性变换都可以由矩阵产生吗？当 T 被描述为旋转或投影等时，总有一个矩阵隐藏在变换 T 下吗？**是的。** 下一节将看到——这一切都是矩阵。一切线性变换都可以用一个矩阵表示。

## 5. 平面的线性变换——房子的例子

看一个变换的效果比定义它有趣多了。用一个有 11 个顶点的房子——有 11 个向量 v，它们变换为 11 个向量 Av。v 之间的直线变成 Av 之间的直线（变换是线性的！）。把 A 作用到一个标准的房子上，可以产生新的房子——可能拉伸或旋转等。

## 6. 总结

1. 一个变换 T 把输入空间的每个 v 变换成输出空间的 T(v)
2. 如果 $T(v+w) = T(v) + T(w)$ 且 $T(cv) = cT(v)$，T 是线性的——直线变换后还是直线
3. **线性组合变换之后还是线性组合：** $T(c_1 v_1 + \cdots + c_n v_n) = c_1 T(v_1) + \cdots + c_n T(v_n)$
4. 只有当 $v_0 = 0$ 时 $T(v) = Av + v_0$ 才是线性的（其实就是 $T(v) = Av$）

## 7. 典型例题

**例 1（偏移变换 / Shear Transformation）：** 消去矩阵 $A = \begin{bmatrix} 1 & 0 \\ 1 & 1 \end{bmatrix}$ 定义了偏移变换，把 $(x,y)$ 变成 $T(x,y) = (x, x+y)$。画出 xy 平面，看看 $(1,0), (1,1)$ 变换成什么。对 x=a 的垂直线发生了什么？如果输入空间是单位正方形 $0 \le x \le 1, 0 \le y \le 1$，画出输出。

**解：** x 轴上的点 $(1,0), (2,0)$ 被 T 变换为 $(1,1), (2,2)$——水平的 x 轴变成逆时针旋转 45 度的直线（经过原点）。y 轴的点没有移动，因为 $T(0,y) = (0,y)$。**y 轴是 $\lambda=1$ 时 T 的特征向量所在的直线。**（A 的特征值是 $\lambda = 1,1$，特征向量是 $x = (0,1)$。）x=a 的直线会向上移动 a。

![](附件/image-20210330101335663.png)

**例 2（非线性变换的可逆性）：** 非线性变换 T，如果输出空间的每个 b 都由输入空间的单独一个 x 变换而来，则是可逆的——$T(x) = b$ 恰好仅有一个解。下面的变换（x 是实数）哪些是可逆的？注意没有一个是线性的，$T_3$ 也不是。求 $T^{-1}$：

1. $T_1(x) = x^2$——$b<0$ 无解，$b>0$ 有两个解 → 不可逆
2. $T_2(x) = x^3$——$x = \sqrt[3]{b}$ 唯一 → 可逆，$T^{-1}(x) = \sqrt[3]{x}$
3. $T_3(x) = x + 9$——$x = b-9$ 唯一 → 可逆，$T^{-1}(x) = x-9$（但不是线性的——因为 $T(0) = 9 \neq 0$）
4. $T_4(x) = e^x$——$b \le 0$ 无解 → 不可逆
5. $T_5(x) = 1/x$（$x \neq 0$）——$x = 1/b$ 唯一 → 可逆
