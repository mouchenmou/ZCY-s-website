!!! abstract
    Hermitian 矩阵（$A^H = A$）是实对称矩阵的复推广——特征值全为实数。酉矩阵（$U^H U = I$）是正交矩阵的复推广——保持长度和内积不变。

---

## 1. Hermitian 矩阵——复对称

实数对称矩阵 $A = A^T$ 的复版本：$A^H = \bar{A}^T = A$。

**性质：**
1. 所有特征值是**实数**（证明和实对称的共轭法完全一样）
2. 特征向量可选为相互正交（按 Hermitian 内积）

**证明特征值为实数：** 设 $Ax = \lambda x$。取 Hermitian：$x^H A^H = \bar{\lambda} x^H$。因 $A^H = A$：$x^H A = \bar{\lambda} x^H$。计算 $x^H A x$ 两种方式：

$$x^H A x = x^H \lambda x = \lambda x^H x, \quad x^H A x = \bar{\lambda} x^H x$$

$x^H x = \sum |x_i|^2 > 0$ → $\lambda = \bar{\lambda}$ → $\lambda$ 是实数。

**例子：**

$$A = \begin{bmatrix} 2 & 3+i \\ 3-i & 5 \end{bmatrix}$$

对角线必须是实数（因为 $\bar{a}_{ii} = a_{ii}$）。非对角元互为共轭：$a_{21} = \overline{a_{12}}$。

---

## 2. 酉矩阵——复正交

正交矩阵 $Q^T Q = I$ 的复版本：$U^H U = I$。**$U^{-1} = U^H$——逆 = 共轭转置。**

**性质：**
1. 所有特征值 $|\lambda| = 1$（在复平面的单位圆上）
2. 保持长度：$\|Ux\| = \|x\|$
3. 保持内积：$(Ux)^H (Uy) = x^H y$
4. 各列是标准正交的（按 Hermitian 内积）：$u_i^H u_j = \delta_{ij}$

酉矩阵可分解为 $U = Q e^{i\Theta}$（正交 × 对角复数相位）。

---

## 3. 对比表

| 实数 | 复数 |
|------|------|
| 转置 $A^T$ | 共轭转置 $A^H$ |
| 对称 $A = A^T$ | Hermitian $A = A^H$ |
| 正交 $Q^T Q = I$ | 酉 $U^H U = I$ |
| $x^T y$ 内积 | $x^H y$ 内积 |
| $\|x\|^2 = x^T x$ | $\|x\|^2 = x^H x$ |
| 特征值可能复数 | Hermitian → 实数；酉 → $|\lambda|=1$ |
