!!! abstract
    正值矩阵的每个元素 $a_{ij}>0$，最大的特征值是正实数，特征向量分量全正。马尔科夫矩阵每列和为 1，$\lambda_{\max}=1$ 对应稳态；人口增长 $\lambda_{\max}>1$；消费矩阵 $\lambda_{\max}<1$。

## 1. 正值矩阵的三种情况

这一节都是关于正值矩阵：每一个 $a_{ij}>0$。关键事实：**它最大的特征值是实数并且是正值，特征向量也是如此。** 在经济学、生态学、人口动态和随机游走中大有帮助：

$$\text{Markov：} \quad \lambda_{\max} = 1, \quad \text{Population：} \quad \lambda_{\max} > 1, \quad \text{Consumption：} \quad \lambda_{\max} < 1$$

---

## 2. 马尔科夫矩阵

在微分方程中，$\lambda=0$ 得到稳态。但对矩阵幂 $u_{k+1}=Au_k$——**$\lambda=1$ 才是最重要的。稳态就是 $\lambda=1$ 对应的特征向量。** 马尔科夫矩阵每列相加为 1，这保证了一个特征值为 1——不需要解 $\det(A-\lambda I)=0$ 就能知道。

**关键：**
1. $\lambda=1$ 是特征值。而且 $x_1$ 的所有分量都 $\ge 0$。初始值为正 → 稳态就是正值
2. 其他所有的特征值 $|\lambda_i|<1$（可能有例外——有些绝对值还是等于 1，但绝不会大于 1）

请记住 $u_k = A^k u_0 = c_1 \lambda_1^k x_1 + \cdots + c_n \lambda_n^k x_n$。要求有一套完整的特征向量，否则 $u_0$ 不能扩展为特征向量形式。如果 $\lambda_1=1$，它取 k 次幂不变，其他 $\lambda$ 小于 1 会随时间趋于 0。**最终，稳态趋向于 $u_0$ 沿着 $x_1$ 的部分 $c_1 x_1$！**

假设对 $A = \begin{bmatrix}.8&.3\\.2&.7\end{bmatrix}$，不断乘正值向量 $u_0 = (a, 1-a)$：

$$u_1 = A u_0, \quad u_2 = A u_1 = A^2 u_0 \tag{L1}$$

k 步后 $u_1, u_2, u_3, \ldots$ 逐渐接近稳态 $u_\infty = (.6, .4)$。而且，**任何 $u_0$ 都会收敛到相同的 $u_\infty$！** 为什么呢？

稳态等式 $A u_\infty = u_\infty$ 使 $u_\infty$ 是 $\lambda=1$ 的特征向量：

$$\begin{bmatrix}.8&.3\\.2&.7\end{bmatrix}\begin{bmatrix}.6\\.4\end{bmatrix} = \begin{bmatrix}.6\\.4\end{bmatrix}$$

再次乘 A 不改变 $u_\infty$。但还没解释为什么所有 $u_0$ 最终都得到 $u_\infty$。其他矩阵也有稳态，但不吸引人：

$$B = \begin{bmatrix}1&0\\0&2\end{bmatrix} \quad \text{也有稳态，但不吸引人：}\quad B\begin{bmatrix}1\\0\end{bmatrix} = \begin{bmatrix}1\\0\end{bmatrix} \tag{L2}$$

对上面的 B，如果输入 $u_0 = (0,1)$，得到 $u_1 = (0,2)$，$u_2 = (0,4)$……第二个分量被加倍了。B 有 $\lambda=1$，但**也有 $\lambda=2$——产生不稳定性**。$\vec{u}$ 沿着不稳定特征向量的分量乘以 $\lambda$，而 $|\lambda|>1$ 意味着增长。

!!! definition "马尔科夫矩阵"
    1. A 的每个元素 $a_{ij} \ge 0$
    2. A 的每一列元素和是 1
    
    **马尔科夫矩阵的幂，还是马尔科夫！**

`Eq(L2)` 的 B 没有性质 2。对于马尔科夫矩阵 A，立刻可得：
1. A 乘非负 $u_0$ → 非负 $u_1$
2. 如果 $u_0$ 的分量加起来是 1，那么 $u_1$ 的分量也是

**第 2 点的原因：** $[1\cdots 1] u_0 = 1$ → $u_0$ 分量和是 1。A 每列和为 1 → 矩阵乘法得 $[1\cdots 1] A = [1\cdots 1]$。再把 $[1\cdots 1]$ 乘 $A u_0$：

$$[1\cdots 1] A u_0 = [1\cdots 1] u_0 = 1$$

所以 $A u_0$ 分量和也是 1。同样 $u_2 = A u_1, u_3 = A u_2$ 也成立。**任何 $A^k u_0$ 都非负且分量和为 1。** 这些都是**概率向量**，它们的极限 $u_\infty$ 也是一个概率向量。

## 3. 例 1：丹佛租车

丹佛出租的车占比 .02，其他地区 .98。每月 80% 留在丹佛、20% 离开；5% 从其他地区进入丹佛、95% 留在原地。

$$\text{First month:}\quad A = \begin{bmatrix}.80&.05\\.20&.95\end{bmatrix}, \quad u_1 = A u_0 = \begin{bmatrix}.80&.05\\.20&.95\end{bmatrix}\begin{bmatrix}.02\\.98\end{bmatrix} = \begin{bmatrix}.065\\.935\end{bmatrix}$$

注意 $.065+.935 = 1$。第二个月：$u_2 = A u_1 = (.09875, .90125)$。所有 $u$ 向量都是正值的（因为 A 是正值的），每个 $u_k$ 的分量和都为 1。那么很长时间后丹佛的车最终占多少比例？

**解：** 对角化 A。A 的特征值是 $\lambda_1=1, \lambda_2=0.75$（迹 = 1.75）。特征向量 $x_1 = (.2, .8), x_2 = (-1, 1)$：

$$A x = \lambda x: \quad A\begin{bmatrix}.2\\.8\end{bmatrix} = 1\begin{bmatrix}.2\\.8\end{bmatrix}, \quad A\begin{bmatrix}-1\\1\end{bmatrix} = .75\begin{bmatrix}-1\\1\end{bmatrix}$$

初始向量 $u_0$ 是 $x_1, x_2$ 的组合，系数为 1、0.18：

$$\text{Combination of eigenvectors:}\quad u_0 = \begin{bmatrix}.02\\.98\end{bmatrix} = \begin{bmatrix}.2\\.8\end{bmatrix} + .18\begin{bmatrix}-1\\1\end{bmatrix}$$

现在 $u_0$ 乘以 A 得到 $u_1$，特征向量分别被 $\lambda_1=1, \lambda_2=0.75$ 乘：

$$u_1 = 1\begin{bmatrix}.2\\.8\end{bmatrix} + (.75)(.18)\begin{bmatrix}-1\\1\end{bmatrix}$$

每个月，都有一个额外的 0.75 乘 $x_2$，而 $x_1$ 不改变。k 步以后：

$$u_k = A^k u_0 = \begin{bmatrix}.2\\.8\end{bmatrix} + (.75)^k(.18)\begin{bmatrix}-1\\1\end{bmatrix}$$

**$\lambda=1$ 的 $x_1$ = 稳态，$x_2$ 因 $|\lambda|<1$ 而消失。** 极限 = $(.2,.8)$——丹佛 20%，其他 80%。就算初始向量是 $u_0=(0,1)$，也可分解为 $u_0 = x_1 + 0.2x_2$，也是趋向于 $u_\infty = x_1$。这就是马尔科夫链的模式。

## 4. 例 V24-1：为什么马尔科夫矩阵一定有 $\lambda=1$？

$A = \begin{bmatrix}.1&.01&.3\\.2&.99&.3\\.7&0&.4\end{bmatrix}$。为什么有 $\lambda=1$？

$$|A - I| = \begin{bmatrix}-.9&.01&.3\\.2&-.01&.3\\.7&0&-.6\end{bmatrix}$$

$A-I$ 相当于平移一个单位矩阵。如果此时奇异 → $\lambda=1$ 是特征值。为什么 $A-I$ 奇异？**各列的元素和是 0！** 每一列都是线性相关 → 行列式为 0 → 奇异。从行向量角度：$(1,1,1)$ 乘 $A-I$ 的各行，全部加起来得 0——它是 $A-I$ 的左零空间 $N((A-I)^T)$ 的向量（方阵行向量线性相关 = 奇异）。

再考虑 $A-I$ 的零空间——列向量的什么组合得到零向量？一旦找到这个向量 $x_1$：$(A-I)x = 0$ → $Ax = x$ → $\lambda = 1$。**这个 $x_1$ 就是稳态！** 计算得 $x_1 = (.6, 33, .7)$，分量都是正值。

**关键：任何 A 和 $A^T$ 的特征值一样。** 为什么？$\det(A-\lambda I) = 0$，而 $\det(A^T-\lambda I) = \det((A-\lambda I)^T) = \det(A-\lambda I)$。所以特征值相同。

$1$ 是 $A^T$ 的特征值，特征向量是 $(1,1,1)$。所以 $1$ 也是 $A$ 的特征值——但特征向量不同。

## 5. 证明马尔科夫矩阵 A 存在 $\lambda_{\max}=1$

如果 A 是**正值的**马尔科夫矩阵（每个 $a_{ij}>0$，每列和为 1），那么 $\lambda_1=1$ 是最大特征值，$x_1$ 是稳态：

$$u_k = x_1 + c_2(\lambda_2)^k x_2 + \cdots + c_n(\lambda_n)^k x_n \quad \text{总是逼近}\quad u_\infty = x_1 \tag{W1}$$

**证明第 1 点：** $\lambda=1$ 是一个特征值。因为 $A-I$ 每一列和都是 $1-1=0$，全部行加起来是 0 → 行线性相关 → $A-I$ 奇异。

**证明第 2 点：** 没有 $|\lambda|>1$。如果存在，$A^k$ 会不断增长。但 $A^k$ 也是马尔科夫矩阵（元素非负，每列和为 1）→ 不会增长！

**例 2（非正规——有 0 元素）：** $A = \begin{bmatrix}0&1\\1&0\end{bmatrix}$，$\lambda=1,-1$。$\lambda_2=-1$ 不衰减——$A^k$ 在 A 和 I 间交替，没有稳态。正规情况（所有元素 $>0$）：$\lambda=1$ 严格大于所有其他特征值，$A^k$ 逼近各列都是稳态的秩一矩阵。

**例 3（3 个小组）：** 每步组 1 的一半去组 2、一半去组 3，其他两组同理。

$$u_1 = A u_0 = \begin{bmatrix}0&1/2&1/2\\1/2&0&1/2\\1/2&1/2&0\end{bmatrix}\begin{bmatrix}p_1\\p_2\\p_3\end{bmatrix}$$

A 有 0 元素，但 $A^2$ 的 0 消失了。$\lambda=1$（稳态 $x_1 = (\frac13,\frac13,\frac13)$），$\lambda_2=\lambda_3=-\frac12$。从 $u_0=(8,16,32)$ 开始：

$$u_0=\begin{bmatrix}8\\16\\32\end{bmatrix}, u_1=\begin{bmatrix}24\\20\\12\end{bmatrix}, u_2=\begin{bmatrix}16\\18\\22\end{bmatrix}, u_3=\begin{bmatrix}20\\19\\17\end{bmatrix}$$

每步总人数 = 56。稳态 = $56\cdot(\frac13,\frac13,\frac13)$。**$|\lambda_2|$ 控制收敛到稳态的速度。**

Google PageRank：网站间链接产生马尔科夫矩阵 A，稳态 $u_\infty$ = 网页排名。

## 6. 例 V24-2：人口迁移

加州和麻省。每年 90% 加州人留在原地、10% 迁麻省；80% 麻省人原地、20% 迁加州。矩阵不变——马尔科夫矩阵。

$$\begin{bmatrix}u_{\text{cal}}\\u_{\text{mass}}\end{bmatrix}_{k+1} = \begin{bmatrix}.9&.2\\.1&.8\end{bmatrix}\begin{bmatrix}u_{\text{cal}}\\u_{\text{mass}}\end{bmatrix}_k$$

A 第一列：加州人 .9 留下、.1 搬去麻省。**矩阵元素 = 留下或迁移的概率，非负且加起来为 1。**

$\lambda_1=1$，$\lambda_2=0.7$（行列式 = 0.7）。$(A-I)x_1=0$ → $x_1=(2,1)$（分量正值）。稳态是 $x_1$ 的倍数。$u_0 = (0,1000)$：

$$u_0 = \begin{bmatrix}0\\1000\end{bmatrix} = \frac{1000}{3}\begin{bmatrix}2\\1\end{bmatrix} + \frac{2000}{3}\begin{bmatrix}-1\\1\end{bmatrix}$$

$$u_{100} = \frac{1000}{3}\begin{bmatrix}2\\1\end{bmatrix} + \frac{2000}{3}(0.7)^{100}\begin{bmatrix}-1\\1\end{bmatrix}$$

$\lambda_2<1$ 部分在未来消失。稳态：人口比 $2:1$，总人口 1000 → 加州 667，麻省 333。

## 7. Perron-Frobenius 定理

只要 $a_{ij}\ge 0$ 就可应用 PF 定理——不需要列和为 1。

!!! theorem "Perron-Frobenius for A > 0"
    $Ax = \lambda_{\max} x$ 中 x 的所有分量都是严格正值的。

**证明思想：** 对一些非负向量 $x \neq 0$，寻找所有满足 $Ax \ge t x$ 的数字 $t$。这些 $t$ 中肯定有一个最大的 $t_{\max}$。对于 $t_{\max}$，证明 $Ax = t_{\max} x$ 等式成立。如果 $Ax \ge t_{\max} x$ 不能取等号，因为 $A>0$，可得严格不等式 $A^2 x > t_{\max} A x$。因此对正值的 $y = Ax$，有 $Ay > t_{\max} y$。

## 8. 人口增长——Leslie 矩阵

三个年龄段：$<20, 20-39, 40-59$。生育率 $F_i$，存活率 $P_i$。20 年后：
- $n_1^{\text{new}} = F_1 n_1 + F_2 n_2 + F_3 n_3$（全部是新生儿）
- $n_2^{\text{new}} = P_1 n_1$（来自 $n_1$ 的存活者）
- $n_3^{\text{new}} = P_2 n_2$

$$
\begin{bmatrix}n_1\\n_2\\n_3\end{bmatrix}^{\text{new}} = \begin{bmatrix}F_1&F_2&F_3\\P_1&0&0\\0&P_2&0\end{bmatrix}\begin{bmatrix}n_1\\n_2\\n_3\end{bmatrix} = \begin{bmatrix}.04&1.1&.01\\.98&0&0\\0&.92&0\end{bmatrix}\begin{bmatrix}n_1\\n_2\\n_3\end{bmatrix}
$$

$A \ge 0$ 但不是 $A>0$。但 $A^3 > 0$，可应用 PF 定理。$\lambda_{\max} = 1.06$。

$$\text{eig}(A) = \begin{array}{r}1.06\\ -1.01\\ -0.01\end{array}, \quad A^2 = \begin{bmatrix}1.08&.05&.00\\.04&1.08&.01\\.90&0&0\end{bmatrix}$$

从 $u_0=(0,1,0)$（只有第二组 1 人）开始：$u_1 = (1.1, 0, .92) = A$ 的列 2。渐进增长率 $\lambda_{\max}$ 对所有起点都一样。特征向量 $x = (.63, .58, .51)$。

## 9. 经济学中的消费矩阵

生产 1 单位产出所需消耗的输入比例 = 消费矩阵 A。设三个部门：化学、食物、石油。

$$A = \begin{bmatrix}.2&.3&.4\\.4&.4&.1\\.5&.1&.3\end{bmatrix}$$

问题：经济能承担对 $y_1, y_2, y_3$ 的需求吗？输入 p，消耗 Ap，净产出 = $p - Ap$。

**求解：** $p - Ap = y$ → $p = (I-A)^{-1} y$。但不仅需要 $(I-A)^{-1}$ 存在，还必须非负——**什么时候 $(I-A)^{-1}$ 是非负矩阵？**

若 A 比 I 小则 Ap < p → 很多产出。取决于 $\lambda_{\max}(A)$（正数）：
- $\lambda_1>1$ → $(I-A)^{-1}$ 有负元素
- $\lambda_1=1$ → 不存在
- **$\lambda_1<1$ → $(I-A)^{-1}$ 非负——这正是我们想要的**

**矩阵几何级数：** 类似 $1/(1-x) = 1+x+x^2+\cdots$（$|x|<1$），当 A 所有 $|\lambda|<1$：

$$(I-A)^{-1} = I + A + A^2 + A^3 + \cdots$$

设此级数为 S。$AS = A+A^2+\cdots$——比 S 少了第一个元素 I。$S - AS = I$ → $(I-A)S = I$ → $S = (I-A)^{-1}$。**这个级数在 A 的所有 $|\lambda|<1$ 时收敛。**
