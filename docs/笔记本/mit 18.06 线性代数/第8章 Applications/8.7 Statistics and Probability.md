!!! abstract
    统计中的线性代数：协方差矩阵、最小二乘、主成分分析（PCA）。$A^T A$ 是数据和最小二乘的核心。

---

## 1. 最小二乘再回顾

$m$ 个数据点，$n$ 个参数（$m > n$）。$Ax = b$ 无解 → 正规方程 $A^T A \hat{x} = A^T b$。

**统计视角：** $b$ 是因变量观测值，$A$ 的列是自变量。$\hat{x}$ 是最佳拟合参数。误差 $e = b - A\hat{x}$ 的平方和 = $\|e\|^2$——最小化噪声。

---

## 2. 协方差矩阵

$X$ 是 $m \times n$ 数据矩阵（$m$ 个样本，$n$ 个变量）。先"中心化"（每列减均值），则 **协方差矩阵 = $\frac{1}{m} X^T X$**。

协方差矩阵是对称半正定的：
- 对角线：每个变量的方差
- 非对角线：两个变量的协方差

**$X^T X$ 的特征值和特征向量 = PCA 的基础。**

---

## 3. 主成分分析（PCA）

寻找数据方差最大的方向。即：找单位向量 $v$，最大化 $\|X v\|^2 = v^T X^T X v$。

约束 $|v|=1$ 下求最大值 → 用拉格朗日乘子 → 解 $X^T X v = \lambda v$。**$v$ 是协方差矩阵的特征向量，对应最大特征值 = 第一主成分方向。**

**PCA 步骤：**
1. 中心化数据 → $X$
2. 计算 $X^T X$ 的特征值和特征向量
3. 按特征值从大到小取前 $k$ 个特征向量 → 降到 $k$ 维
4. 新坐标 = $X v_1, \ldots, X v_k$

这等价于 $X$ 的 SVD：$X = U\Sigma V^T$。$V$ 的列 = 主成分方向，$\Sigma$ 的对角元 = $\sqrt{\lambda_i}$。

---

## 4. 马尔科夫链和 PageRank

转移矩阵 $P$（$P_{ij}$ = 从状态 $j$ 到 $i$ 的概率）。稳态分布 $\pi$ 满足 $\pi = P\pi$——$\pi$ 是 $P$ 的 $\lambda=1$ 的特征向量。

Google PageRank：网页 = 状态，链接 = 转移。$\pi_i$ = 第 $i$ 页的最终排名。
