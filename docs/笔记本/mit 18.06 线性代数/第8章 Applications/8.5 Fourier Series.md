!!! abstract
    傅里叶级数：把周期函数展开到正交基（sin、cos）上。函数变成向量，点乘变成积分。sin 和 cos 在 $[0,2\pi]$ 上正交——无穷维希尔伯特空间的完美例子。

---

## 1. 从标准正交基到傅里叶级数

先讨论带有标准正交基的投影问题。基向量是 $q_1,q_2,\ldots,q_n$，n 个标准正交向量，是 n 维空间的一组完整基，任何向量 v 都可以由这组基构成：

$$v = x_1 q_1 + x_2 q_2 + \cdots + x_n q_n$$

那么这些组合系数是多少？比如 $x_1$ 是多少？**这就是投影——将向量展开到基上去。** 这组基的特别之处在于标准正交，从而系数 x 就有特殊的公式。

怎么求出 $x_1$？可以根据投影矩阵 $Q^T Q$ 或正规方程。但怎么才能去除其他的 x 值，只保留 $x_1$？其实早就学过——将每一项都和 $q_1$ 做点乘。最后一项 $x_n q_n$ 和 $q_1$ 点乘结果是 0（因为 q 向量是标准正交的！）：

$$q_1^T v = x_1 q_1^T q_1 + 0 + \cdots + 0 = x_1$$

立马就可以得到 $x_1$！矩阵形式：

$$\underbrace{\begin{bmatrix} | & \cdots & | \\ q_1 & \cdots & q_n \\ | & \cdots & | \end{bmatrix}}_{Q} \begin{bmatrix}x_1\\\vdots\\x_n\end{bmatrix} = v \tag{B1}$$

即 $Qx = v$。解是 $x = Q^{-1} v$。重点：**Q 的逆很简单——就是 $Q^T$！** 所以 $x = Q^{-1} v = Q^T v$，$x_1$ 就是 $Q^T$ 的第一行乘 v，也就是 $q_1^T v$。这里的关键是 q 向量是标准正交的，**这也是傅里叶级数建立的基础。**

## 2. 函数空间：向量变成函数

已知函数 $f(x)$，想把这个函数写成组合的形式。应该有一个常数项，包含 $\cos x, \sin x$，还有 $\cos 2x, \sin 2x$ 等等：

$$f(x) = a_0 + a_1 \cos x + b_1 \sin x + a_2 \cos 2x + b_2 \sin 2x + \cdots \tag{B2}$$

这和 Eq(B1) 有什么不同？这个是**无穷维**的，但关键性质还是**正交**！$\sin$ 和 $\cos$ 是正交的！这使得傅里叶级数有意义。Eq(B2) 的 $f(x)$ 就是傅里叶级数，可以作用在函数空间上。对比 Eq(B1)：

- 向量 v 变成函数 $f(x)$
- 正交向量 $q_1,\ldots,q_n$ 变成正交函数：无穷多个的常数 1（系数 $a_0$ 对应的函数）、$\sin x, \cos x, \sin 2x, \cos 2x, \ldots$
- 空间是无穷维的而不是有限维

向量现在是函数！基向量是函数：$1, \cos x, \sin x, \cos 2x, \sin 2x, \ldots$。傅里叶级数成立的原因就是因为这些函数是正交的！

**函数的点乘是什么？** 向量正交是点积为 0：$v^T w = v_1 w_1 + v_2 w_2 + \cdots + v_n w_n = 0$。对于函数来讲，给定两个函数 f, g。向量有 n 个元素，但函数是连续的——画函数图，不是画 n 个点，而是整个连续的曲线。$f^T g$ 是什么？想要和向量的尽量类似，最相似的就是在每个 x 值的 $f(x) \cdot g(x)$，都类似于 $v_1 w_1$。向量那里是 n 个乘积组成和，但函数是一系列的 x，对应的和是什么？**在连续的情况下，加法对应的是什么——就是积分！**

$$f^T g = \int f(x) g(x) dx$$

再看积分上下限。Eq(B2) 的右边部分都是周期函数，周期是 $2\pi$，而 $f(x)$ 最好也是这样，所以 $f(x)$ 的积分上下限就是 $0$ 到 $2\pi$。$f(x)$ 必须也是周期的，也就是 $f(x) = f(x+2\pi)$。检查一下正交性：

$$\int_0^{2\pi} \sin x \cos x \, dx = \left. \frac12 \sin^2 x \right|_0^{2\pi} = 0$$

现在一切就绪！存在一个向量空间，但现在向量是函数，其内积定义为积分，这样就有了正交性。现在有了函数空间的无穷正交基，我们需要的做就是把函数展开到基上。来看看 $\cos x$ 的系数 $a_1$ 是多少。常数项 $a_0$ 很容易，是 f 的平均值，讨论 $a_1$ 更加典型。怎么求？就像向量的情况一样求——**每一项都和 $\cos x$ 取内积！**

Eq(B2) 的左边变成 $\int_0^{2\pi} f(x)\cos x\,dx$，右边只有 $a_1\cos x$ 与 $\cos x$ 的积分不是 0，也就是 $a_1 \int_0^{2\pi} \cos^2 x \, dx = a_1 \pi$。最终得到：

$$\int_0^{2\pi} f(x)\cos x \, dx = a_1 \underbrace{\int_0^{2\pi} \cos^2 x \, dx}_{\pi}$$

$$a_1 = \frac{1}{\pi} \int_0^{2\pi} f(x)\cos x \, dx$$

**这就是著名的傅里叶级数系数公式，可见的确可以扩展到一组标准正交基上。**

## 3. 无限维下的新问题

本节将会解释线代在无限维下也是可以起作用的，但首先要将一些概念做适当的改变。

**一个向量有无穷分量是什么意思？** 有两个不同的答案：
1. 向量变成 $v = (v_1, v_2, \ldots)$，无穷个分量
2. 向量是一个函数，如 $\sin x$

两个答案都是可以的，傅里叶级数把它们联系起来。

**点乘的定义。** 无穷分量向量的点乘是一个**无穷级数**：

$$v \cdot w = v_1 w_1 + v_2 w_2 + \cdots \tag{1}$$

这就产生了一个不会在 $R^n$ 下出现的问题：**这个级数收敛到一个有限数字吗？** 这就是无穷维和有限维的最大区别。

明显，当 $v = w = (1,1,\ldots)$，级数不收敛，点乘得到的和是无限的。这里 $v=w$，所以实际上计算的是 $v\cdot v = |v|^2 =$ 长度平方，而向量 $(1,1,\ldots)$ 的长度是无限的。这不是我们需要的向量——我们唯一允许的向量是长度有限的向量。

!!! definition "希尔伯特空间"
    当且仅当向量 $v = (v_1,v_2,\ldots)$ 的长度有限时，它才属于无穷维的**希尔伯特空间**。即：
    $$\|v\|^2 = v \cdot v = v_1^2 + v_2^2 + v_3^2 + \cdots$$
    必须加和为有限数字。

**例 1：** 向量 $v = (1, \frac12, \frac14, \ldots)$ 在希尔伯特空间内，因为它自身点乘是一个 $4/3$ 的几何级数，长度是 $\sqrt{4/3}$：

$$v \cdot v = 1 + \frac14 + \frac{1}{16} + \cdots = \frac{1}{1 - 1/4} = \frac43$$

现在问题是：如果 v, w 都是有限长度的，它们的点乘会不会是无限的？不会。$v \cdot w = v_1 w_1 + v_2 w_2 + \cdots$ 一定是有限数字。证明很简单——柯西不等式：

$$|v \cdot w| \le \|v\| \|w\|$$

而且 $v\cdot w$ 所占 $|v||w|$ 的比例仍然是 v, w 夹角 $\theta$ 的 $\cos$ 值。就算在无穷维下，$|\cos\theta|$ 也不会大于 1。

## 4. 函数作为向量：内积和长度

现在看把向量定义为函数的方式。这些函数都是向量！定义在 $0 \le x \le 2\pi$ 上的函数 $f(x), g(x), \ldots$ 比 $R^n$ 空间更大。这时 f(x), g(x) 的点乘是什么？f(x) 的长度是什么？

在连续的情况下，关键就是：**求和变为求积分！** 现在点乘不是 $v_j \cdot w_j$ 的和，而是 $f(x) \cdot g(x)$ 的积分。此时点乘称为内积：

!!! definition "函数的内积和长度"
    函数 $f(x), g(x)$ 的内积：
    $$(f,g) = \int_0^{2\pi} f(x) g(x) dx \tag{3.1}$$
    长度平方：
    $$\|f\|^2 = \int_0^{2\pi} (f(x))^2 dx \tag{3.2}$$
    函数的定义域 $[0,2\pi]$ 是因为下一个例子将是 $\cos x, \sin x$。

**例 2：** $f(x) = \sin x$ 的长度来自它自身的内积：

$$(f,f) = \int_0^{2\pi} (\sin x)^2 dx = \pi \quad \Rightarrow \quad \text{length of } \sin x = \sqrt{\pi}$$

以上积分可将 $\sin^2 x$ 改写为 $1/2 - 1/2 \cos 2x$，它的值在 $1/2$ 上下浮动，平均值就是 $1/2$，再乘区间长度 $2\pi$，得到结果 $\pi$。

**重要：$\sin x$ 和 $\cos x$ 在函数空间下是正交的：**

$$\int_0^{2\pi} \sin x \cos x \, dx = \int_0^{2\pi} \frac12 \sin 2x \, dx = \left[-\frac14 \cos 2x\right]_0^{2\pi} = 0 \tag{4}$$

**这个 0 不是巧合！** 而且这个正交性不仅仅是 $\sin x$ 和 $\cos x$ 两个函数。无穷长度的 sin、cos 都是正交的，包括 $\cos 0x(=1), \sin x, \cos x, \sin 2x, \cos 2x, \sin 3x, \cos 3x, \ldots$——此列表的任何一个函数都和其他函数正交。

## 5. 傅里叶级数的能量关系

一个函数的傅里叶级数是根据 sin 和 cos 展开的：

$$y(x) = a_0 + a_1 \cos x + b_1 \sin x + a_2 \cos 2x + b_2 \sin 2x + \cdots \tag{5}$$

那些 sin、cos 就是正交基，这些基位于函数空间下。这些基是周期的——在 $[2\pi, 4\pi]$ 区间，表现和 $[0,2\pi]$ 完全一样。

注意：Eq(5) 是无限扩展的，傅里叶级数是无穷级数。前面我们避开了类似 $v = (1,1,\ldots)$ 这样长度无限的向量。现在还要避开类似 $\frac12 + \cos x + \cos 2x + \cos 3x + \cdots$ 这样的函数——因为它是 $\pi$ 乘以著名的 $\delta(x)$（delta 函数）。在 $x=0$ 它的长度是 $\frac12 + 1 + 1 + 1 + \cdots$ 是无穷的；在 $0 < x < 2\pi$ 的其他所有点，这个级数以某种方式平均地加到 0。$\int \delta(x) = 0$，但 $\int \delta^2(x) = \infty$，所以 delta 函数是被希尔伯特空间排除的。

**计算典型 f(x) 的长度平方：** 由正交性，所有类似 $\cos x \cos 2x$ 乘法的积分都是 0，只留下 sin、cos 平方自身的积分：

$$\|f\|^2 = 2\pi a_0^2 + \pi(a_1^2 + b_1^2 + a_2^2 + b_2^2 + \cdots) \tag{6}$$

第 1 行到第 2 行是因为正交性；第 3 行计算了这个积分的值。如果将每个函数都除以自身的长度，就得到了**标准正交基：**

$$\frac{1}{\sqrt{2\pi}}, \frac{\cos x}{\sqrt{\pi}}, \frac{\sin x}{\sqrt{\pi}}, \frac{\cos 2x}{\sqrt{\pi}}, \frac{\sin 2x}{\sqrt{\pi}}, \ldots \quad \color{orange}\text{是函数空间的标准正交基}$$

上面都是单位向量。用系数 $A_0, A_1, B_1, A_2, \ldots$ 来组合它们得到函数 $F(x)$，则 Eq(6) 的 $2\pi, \pi$ 可以去掉了：

$$\|F\|^2 = A_0^2 + A_1^2 + B_1^2 + A_2^2 + B_2^2 + \cdots \tag{7}$$

**重要：函数长度 = 系数向量长度。** 当且仅当系数向量是无限长度的时候，函数有无限的长度（The function has finite length exactly when the vector of coefficients has finite length）。傅里叶级数给出了函数空间 $L^2$ 和无穷维希尔伯特空间 $\ell^2$ 的完美匹配。仅当希尔伯特空间包含了作为傅里叶系数的向量 $v = (a_0, a_1, b_1, \ldots)$，函数空间才包含 $f(x)$。$f(x)$ 和 v 都有有限的长度。
