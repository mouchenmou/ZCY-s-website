# 1.测量不确定度与误差
- 不确定度表示由于测量误差存在而对被测量值不能确定的程度。不确定度是一定概率下的误差限值。

- 不确定度反映了可能存在的误差分布范围，即随机误差分量和未定系统误差的联合分布范围。

- 由于真值的不可知，**误差一般是不能计算的，它可正可负也肯能十分接近零**；而**不确定度总是不为零的正值**，是可以具体评定的。
