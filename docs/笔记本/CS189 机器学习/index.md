# CS189 机器学习

UC Berkeley CS189: Introduction to Machine Learning，伯克利最热门最难抢的课程之一，比纯 CS 的 ML 课更偏数学推导，深入讲概率论、线性代数、优化理论在 ML 中的底层原理。

Jonathan Shewchuk 教授300+页的 Notes 简直就是机器学习的百科全书，从最小二乘一路推到 SVM、神经网络、PCA、贝叶斯，每个公式都手把手推导。

这门课适合想真正理解为什么梯度下降能收敛、为什么 SVM 的 hinge loss 长那样、为什么 PCA 等价于 SVD等等知识点的人。
