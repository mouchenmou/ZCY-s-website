# 信号与系统

强烈推荐胡浩基老师的b站，本人每次上课都是坐在最后一排2倍速看胡浩基的b站视频的。跟着胡浩基真的是能学到非常非常非常多的知识，感觉上完他的这一套课我的数学能力都提升了一个维度。

信号与系统真的很容易被上成只记结论不懂原理的课，这样我真的觉得毫无意义，单纯就是应付考试考完就忘记，那学这门课的意义何在呢？我电院的高中同学，他们上的是信号分析与处理，他们的课程进度就很不合理，我们连连续时间系统的傅里叶变换都还没讲完他们就已经把FFT都上完了，这样子真的就是纯纯背公式但是不知道为什么是这么来的。

所以本人强推胡浩基，他真的是把每一个知识点都讲的非常非常到位，是我在大学期间上过的所有课程中我认为讲的最好的老师，没有之一。
