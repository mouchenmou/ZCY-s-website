在写4-30这道题的时候，用到了一个很巧妙的方法：**黄金定律。**

其学术上的正式名称是：**线性时不变（LTI）系统的特征函数性质**

### 一、 核心定义

如果一个离散时间系统是线性时不变（LTI）的，且其输入是一个复指数信号 $x[n] = e^{j\omega n}$（其中 $\omega$ 是任意实数频率），那么系统的输出 $y[n]$ 将是该输入信号本身乘以一个复常数 $H(e^{j\omega})$。

用数学语言表达即为：

- **输入：** $x[n] = e^{j\omega n}$
- **输出：** $y[n] = H(e^{j\omega}) e^{j\omega n}$

其中，比例系数 $H(e^{j\omega})$ 称为系统在频率 $\omega$ 处的**频率响应**。

物理意义：复指数信号是 LTI 系统的“特征函数”。当它穿过系统时，它不会产生任何新的频率分量（不会变形），系统唯一能对它做的事，就是通过复常数 $H(e^{j\omega})$ 改变它的幅度（也就是 $H(e^{j\omega})$ 的模）和相位（也就是 $H(e^{j\omega})$ 的幅角）。


### 二、 数学证明

这个性质的证明非常直接，只需要利用 LTI 系统的时域卷积性质

**已知条件：**

1. 设 LTI 系统的单位脉冲响应为 $h[n]$。
2. 任意输入 $x[n]$ 与输出 $y[n]$ 的关系由卷积和决定：
    
    $$y[n] = x[n] * h[n] = \sum_{k=-\infty}^{\infty} h[k] x[n-k]$$
    

**证明过程：**

**步骤 1：代入特定的输入信号**

将复指数输入信号 $x[n] = e^{j\omega n}$ 代入上述卷积公式中：

$$y[n] = \sum_{k=-\infty}^{\infty} h[k] e^{j\omega (n-k)}$$

**步骤 2：利用指数运算性质展开**

将括号内的指数部分拆开：

$$y[n] = \sum_{k=-\infty}^{\infty} h[k] \left( e^{j\omega n} \cdot e^{-j\omega k} \right)$$

**步骤 3：提取与求和变量无关的项**

观察求和符号 $\sum$，求和变量是 $k$。这意味着对于这个求和过程而言，包含 $n$ 的项 $e^{j\omega n}$ 只是一个常数，可以将其提取到求和符号的外面：

$$y[n] = e^{j\omega n} \left( \sum_{k=-\infty}^{\infty} h[k] e^{-j\omega k} \right)$$

**步骤 4：识别频率响应**

观察括号内的求和式：

$$\sum_{k=-\infty}^{\infty} h[k] e^{-j\omega k}$$

这正是系统单位脉冲响应 $h[n]$ 的离散时间傅里叶变换（DTFT）的定义式！我们通常将其记为 $H(e^{j\omega})$，即系统的频率响应。

将其代换回步骤 3 的等式中，即得到最终结论：

$$y[n] = H(e^{j\omega}) e^{j\omega n}$$

因此我们不需要考虑 $H(jw)\ 转化为\ H(e^{jw})$ 时要不要乘以 $T$ 。因为重建的第二步中滤波器一定会给它乘上的
