## 收获1：为什么收敛域总是限制实部而不限制虚部

这个收获的灵感来自书本的6-10这道题。
![](附件/Pasted%20image%2020260528234351.png)
这道题的 $X(s)$ 与 $Y(s)$ 的 表达式很好求，计算收敛域的时候引发了我的一个问题：它们的极点算出来是 $\pm2j$ ，也就是实部为0，又因为是右边信号，所以收敛域为 $Re(s)>0$ ，但是为什么只对实部有限制而对虚部没限制？

### 解答：
拉普拉斯变换的定义式是：

$$X(s) = \int_{-\infty}^{\infty} x(t) e^{-st} dt$$

我们把复频率展开，令 $s = \sigma + j\omega$（其中 $\sigma$ 是实部，$\omega$ 是虚部），代入积分中：

$$X(\sigma + j\omega) = \int_{-\infty}^{\infty} x(t) e^{-(\sigma + j\omega)t} dt = \int_{-\infty}^{\infty} \left[ x(t) e^{-\sigma t} \right] e^{-j\omega t} dt$$

要让这个积分有意义（即收敛），我们需要满足绝对可积条件：

$$\int_{-\infty}^{\infty} \left| x(t) e^{-\sigma t} e^{-j\omega t} \right| dt < \infty$$

这里来到了最关键的一步！根据欧拉公式，$e^{-j\omega t} = \cos(\omega t) - j\sin(\omega t)$。在复数中，**无论 $\omega$ 和 $t$ 是多少，这个纯虚指数的模长永远等于 1**：

$$\left| e^{-j\omega t} \right| = \sqrt{\cos^2(\omega t) + (-\sin(\omega t))^2} = 1$$

所以，上面的收敛条件就化简成了：

$$\int_{-\infty}^{\infty} \left| x(t) e^{-\sigma t} \right| \cdot 1 dt < \infty$$

**看到了吗？$\omega$ 被完全消去了！** 积分能不能收敛，100% 只取决于信号 $x(t)$ 乘以衰减因子 $e^{-\sigma t}$ 之后，其面积是否有限。因此，收敛域**只和实部 $\sigma$ 有关**。

## 收获2：Bedrosian 定理
虽然希尔伯特变换不考，但是理解这个地方的内容还是蛮有意思的，引入书本第5-27这道题来讲解。
![](附件/Pasted%20image%2020260528144944.png)
### 1. 定理核心表述

对于两个信号：低频信号 $m(t)$ 和高频载波信号 $c(t)$，如果它们的频谱在频域上完全不重叠（即满足**窄带条件**），则它们乘积的希尔伯特变换，等于低频信号乘以高频信号的希尔伯特变换：

$$\text{H}[m(t)c(t)] = m(t)\text{H}[c(t)]$$

在通信原理的经典窄带信号模型中，设载波为 $c(t) = \cos(\omega_0 t)$，则定理具体体现为：

$$\text{H}[m(t)\cos\omega_0 t] = m(t)\sin\omega_0 t$$

### 2. 前置数学准备

证明该定理，我们需要从频域（傅里叶变换）入手，利用希尔伯特变换在频域的特性。

**① 希尔伯特变换的频域定义**

对信号做希尔伯特变换，等价于使其通过一个冲激响应为 $\frac{1}{\pi t}$ 的系统。其频域传递函数 $H(\omega)$ 是一个纯相移器：

$$H(\omega) = -j \operatorname{sgn}(\omega)$$

即：对所有正频成分（$\omega > 0$）相移 $-90^\circ$（乘 $-j$），对所有负频成分（$\omega < 0$）相移 $+90^\circ$（乘 $+j$）。

**② 信号的频移性质**

设 $m(t) \xleftrightarrow{\mathcal{F}} M(\omega)$，根据傅里叶变换的频移特性，调幅信号 $x(t) = m(t)\cos\omega_0 t$ 的频谱为：

$$X(\omega) = \mathcal{F}\left[m(t) \frac{e^{j\omega_0 t} + e^{-j\omega_0 t}}{2}\right] = \frac{1}{2}M(\omega - \omega_0) + \frac{1}{2}M(\omega + \omega_0)$$

### 3. 严格推导过程

**第一步：确立窄带条件（最核心前提）**

设低频信号 $m(t)$ 的最高频率限制为 $W$。

由于是**窄带系统**，高频载波的频率 $\omega_0$ 必然远大于低频信号的带宽，即满足：

$$\omega_0 > W$$

**第二步：在频域施加希尔伯特变换**

对乘积信号 $x(t)$ 做希尔伯特变换，在频域表示为：

$$\mathcal{F}\{\text{H}[x(t)]\} = \hat{X}(\omega) = -j \operatorname{sgn}(\omega) \cdot X(\omega)$$

代入 $X(\omega)$ 的展开式：

$$\hat{X}(\omega) = -j \operatorname{sgn}(\omega) \left[ \frac{1}{2}M(\omega - \omega_0) + \frac{1}{2}M(\omega + \omega_0) \right]$$

**第三步：利用窄带条件分析符号函数 $\operatorname{sgn}(\omega)$**

此时，我们需要分析方括号内两项所在的频率区间：

1. **对于 $M(\omega - \omega_0)$ 项**：
    
    该频谱分布在区间 $(\omega_0 - W, \omega_0 + W)$。
    
    因为 $\omega_0 > W$，所以这个区间内的所有角频率 $\omega$ **恒大于 0**。
    
    因此，在此区间内，$\operatorname{sgn}(\omega) \equiv 1$。
    
2. **对于 $M(\omega + \omega_0)$ 项**：
    
    该频谱分布在区间 $(-\omega_0 - W, -\omega_0 + W)$。
    
    因为 $\omega_0 > W$，所以这个区间内的所有角频率 $\omega$ **恒小于 0**。
    
    因此，在此区间内，$\operatorname{sgn}(\omega) \equiv -1$。
    

**第四步：化简频域表达式**

将 $\operatorname{sgn}(\omega)$ 的分段常数值代入原式，去括号：

$$\hat{X}(\omega) = -j(1) \cdot \frac{1}{2}M(\omega - \omega_0) - j(-1) \cdot \frac{1}{2}M(\omega + \omega_0)$$

$$\hat{X}(\omega) = -j \frac{1}{2}M(\omega - \omega_0) + j \frac{1}{2}M(\omega + \omega_0)$$

提取公因式 $\frac{1}{2j}$ （由于 $-j = \frac{1}{j}$ 且 $j = -\frac{1}{-j}$）：

$$\hat{X}(\omega) = \frac{1}{2j} M(\omega - \omega_0) - \frac{1}{2j} M(\omega + \omega_0)$$

整理得：

$$\hat{X}(\omega) = \frac{M(\omega - \omega_0) - M(\omega + \omega_0)}{2j}$$

**第五步：通过逆傅里叶变换回到时域**

我们反观第四步得出的频域表达式 $\hat{X}(\omega)$，它恰好是时域信号 $m(t)\sin\omega_0 t$ 的傅里叶变换形式。

利用欧拉公式 $\sin\omega_0 t = \frac{e^{j\omega_0 t} - e^{-j\omega_0 t}}{2j}$ 进行逆向推导：

$$\mathcal{F}[m(t)\sin\omega_0 t] = \mathcal{F}\left[ m(t) \frac{e^{j\omega_0 t} - e^{-j\omega_0 t}}{2j} \right] = \frac{M(\omega - \omega_0) - M(\omega + \omega_0)}{2j}$$

两端对比完全一致。因此，对其求逆傅里叶变换即得证：

$$\text{H}[m(t)\cos\omega_0 t] = m(t)\sin\omega_0 t$$

### 4. 结论总结

证明的关键在于：高频载波将低频包络的频谱完美地“推”到了频域的正、负半轴两端。由于两者在频域不重叠（窄带条件），希尔伯特滤波器的阶跃特性 $-j \operatorname{sgn}(\omega)$ 才能精准地对两团频谱分别进行 $+90^\circ$ 和 $-90^\circ$ 的相移，从而只改变了载波的相位，而将低频信号作为常数系数保留了下来。
