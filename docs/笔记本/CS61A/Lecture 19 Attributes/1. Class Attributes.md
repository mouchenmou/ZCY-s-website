## Assignment to Attributes
Assignment statements with a dot expression on their left-hand side affect attributes for the object of that dot expression（这句话的意思是，当你写出 `A.B = C` 这种代码时，点号左边的那个 `A`，就是你要修改或添加属性的目标。代码只会去改变 `A` 的肚子里的东西。）

- If the object is an instance, then assignment sets an instance attribute
- If the object is a class, then assignment sets a class attribute
