## Bound Method
- All objects have attributes, which are name-value pairs.
    - 无论是类还是实例，它们在 Python 底层其实都像是一个“字典”，里面装着一对对的名字（Name）和具体的值（Value）。
    - Name: `'balance'` -> Value: `0`
    - Name: `'holder'` -> Value: `'Tom'`
- A class is a type (or category) of objects
- Classes are objects too, so they have attributes
    - `Account` 这个类本身也是个对象。它身上绑定的属性有什么呢？
     - Name: `'interest'` -> Value: `0.02`
    - Name: `'deposit'` -> Value: `<function deposit...>`
    - 可以像操作普通对象一样，对类进行赋值操作：`Account.interest = 0.04`
- Instance attribute: attribute of an instance
    - `tom_account.balance` 是实例属性
- Class attribute: attribute of the class of an instance
    - `Account.interest` 是类属性

## Methods and Functions
Python distinguishes between:

- Functions, which we have been creating since the beginning of the course.
- Bound methods, which couple together a function and the object on which that method will be invoked.
