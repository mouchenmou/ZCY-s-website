## 错误方法二：
```Python
    >>> para1+=para[0]
    >>> para1[0]
    # 'h' //终端的输出结果
```

### 错误原因
它虽然没有语法错误，但是只输出了h
- **第一步**：Python 提取 `para[0]`，它的值是字符串 **`'hi'`**。
- **第二步**：代码变成了 `para1 += 'hi'`。
- **第三步**：
    - 对于列表来说，`+=` 的意思是：把等号右边的那个集合里的每一个元素，一个接一个地拿出来，拼接到左边的列表后面。
    - 虽然 `'hi'` 是一个单词，但在 Python 眼里，它也是一个字符的集合：包含 `'h'` 和 `'i'`。
    - 所以，此时 `para1` 变成了：`['h', 'i']`

---

## 正确做法：

### 使用append函数：

```python
    para1 = []
    para1.append(para[0]) 
    # 结果: ['hi']
```
