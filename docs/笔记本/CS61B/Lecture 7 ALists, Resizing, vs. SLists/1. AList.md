## Naive AList Code
```java
    public class AList{
        private int[] items;
        private size;
        
        public AList(){
            items = new int[100]
            size = 0;
        }
        
        public void addLast(int x){
            items[size] = x;
            size + =1;
        }
        
        public int getLast(){
            return items[size-1];
        }
        
        public int get(int i){
            return items[i];
        }
        
        public int size(){
            return size;
        }
        
        public int removeLast(){
            int x = getLast();
            items[size-1] = 0; //这一步可有可无，不影响
            size = size - 1;
            return x;
        }
    }
```

### AList Invariants:
- The position of the next item to be inserted is always size.
- size is always the number of items in the AList.
- The last item in the list is always in position size-1
