当 `size == items.length` 时，我们再往item中添加新的元素会出现错误，那我们就重新创建一个数组，然后把原来数组的内容都复制进来，然后让item指向这个新的数组，再将原先的数组删掉。假设`size`已经等于`items.length`了，执行addLast(11)的过程如下：

- `int[] a = new int[size+1]`
- `System.arraycopy(...)`
- `a[size] = 11;`
- `items = a;`
- `size + =1;`

我们并没有对原先的item做任何改动，只是创建了一个副本，并将`item`指针指向了这个新的数组。


### 为了实现Resizeing，我们可以这么做：
#### 方法一：对addLast这个函数做一下改动：
```java
    public void addLast(int x){
        if(size == items.length){
            int a[] = new int[size+1];
            System.arraycopy(items, 0, a, 0, size);
            a[size] = x;
            size = size + 1;
            items = a;
        }
        itemsp[size] = x;
        size = size + 1;
    }
```

#### 方法二：创建一个private的resize函数
```java
    private void resize(int capacity){
        int a[] = new int[capacity];
        System.arraycopy(items, 0, a, 0, size);
        items = a;
    }
    
    public void addLast(int x){
        if(size == items.length){
            resize(size + 1);
        }
        items[size] = x;
        size = size + 1;
```

方法二会比方法一更好，因为方法二可以独立测试addLast和resize两个函数，通过将代码分解成小块，可以让维护和理解变得更加容易。

## Resizing Slowness

![](附件/Pasted%20image%2020260315154912.png)

这种方式会执行的很慢，因为size超过了容量之后，每次添加元素都要重新把前面的所有元素都复制一遍，当size很大很大的时候，这样实在是太慢了，因此有个改进的方法：

我们不在执行 `size+1`，而是执行 `size*2` , `size*3` 等操作
```java
    private void resize(int capacity){
        int a[] = new int[capacity];
        System.arraycopy(items, 0, a, 0, size);
        items = a;
    }
    
    public void addLast(int x){
        if(size == items.length){
            resize(size * 2);
        }
        items[size] = x;
        size = size + 1;
```

## Genertic AList

### 注意⚠：Java不允许泛型数组的存在

```java
    public class AList<Item>{
        private Item[] items;
        private size;
        
        public AList(){
        //Java不允许泛型数组的存在，因此items = new Item[100];会报错
            items = (Item[]) new Object[100];
            size = 0;
        }
        
        public Item getLast(){
            return items[size-1];
        }
        
        private void resize(int capacity){
        //与AList同理
            Item a[] = (Item[]) new Object[capacity];
            System.arraycopy(items, 0, a, 0, size);
            items = a;
        }
    
        public void addLast(Item x){
            if(size == items.length){
                resize(size * 2);
            }
            items[size] = x;
            size = size + 1;
        }
        
        public Item get(int i){
            return items[i];
        }
        
        public int size(){
            return size;
        }
        
        public int removeLast(){
            Item x = getLast();
            items[size-1]=null; //为了不浪费内存
            size = size - 1;
            return x;
        }
    }
```
