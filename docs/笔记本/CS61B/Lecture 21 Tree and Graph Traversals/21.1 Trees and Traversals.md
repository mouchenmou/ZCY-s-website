## Tree Traversal Orderings

Level Order：

- Visit top-to-bottom, left-to-right (liking reading in English): DBFACEG

Depth First Traversals（深度优先遍历）

- 3 types: Preorder, Inorder, Postorder
- Basic (rough) idea: Traverse "deep nodes"（e.g.A）before shallow ones（e.g.F）
- Note: Traversing a node is different than "visiting" a node. 
![](附件/Pasted%20image%2020260501215938.png)


- PreOrder：中 左 右
- InOrder：左 中 右
- PostOrder：左 右 中
```c
    PreOrder(BSTNode X){
        if(X==NULL) return;
        print(X.key);
        PreOrder(X.left);
        PreOrder(X.right)
    }
    
    InOrder(BSTNode X){
        if(X==NULL) return;
        InOrder(X.left);
        print(X.key);
        InOrder(X.right)
    }
    
    PostOrder(BSTNode X){
        if(X==NULL) return;
        PostOrder(X.left);
        PostOrder(X.right);
        print(X.key);
    }
```
