之前学过了，快速回顾：

- **分**：把数组分成大致两半
- **治**：递归排序每一半
- **合**：合并两个有序半 → 最终结果

[这是一个完整的演示过程](https://docs.google.com/presentation/d/1h-gS13kKWSKd_5gt2FPXLYigFY4jf5rBkNFl3qZzRRw/edit)

|     |                    |
| --- | ------------------ |
| 最好  | $\Theta(N \log N)$ |
| 最坏  | $\Theta(N \log N)$ |
| 空间  | $\Theta(N)$（辅助数组）  |

归并排序也能原地做，但算法非常复杂，且常数因子很大，实际不用。
