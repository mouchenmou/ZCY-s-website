Selection Sort就是最无脑的排序，也是我学C语言的时候最喜欢用的无脑排序法，想看一遍实现过程的话可以[看看这个链接](https://docs.google.com/presentation/d/1p6g3r9BpwTARjUylA0V0yspP2temzHNJEJjCG41I4r0/edit?slide=id.g463de7561_042#slide=id.g463de7561_042)：

1. 找到当前未排序的最小的元素
2. 把它换到最前面
3. 重复上面两个步骤


```
for (int i = 0; i < n - 1; i++) { 
    int min = i; 
    for (int j = i + 1; j < n; j++){
        if (arr[j] < arr[min_idx]) { 
            min = j;
        }
    }
}
```
