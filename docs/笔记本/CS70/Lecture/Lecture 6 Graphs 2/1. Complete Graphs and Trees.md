## 1. Complete Graphs
### 1.1 定义：
一个 **无向图** 如果满足任意两个不同的顶点之间都有一条边相连，那么这个图就叫做我完全图。记作 $K_n$ ，表示一个有n个顶点的完全图。
![](附件/Pasted%20image%2020260426184356.png)
## 1.2 性质：
#### 1.2.1 边数公式：
如果一个Complete Graph有n个顶点，那么这幅图的边数为 $\frac{n(n-1)}{2}$。

就好比n个人他们互相跟所有人都握过手，那么每个人都握了 $(n-1)$ 次，总共有 $n$ 个人，加起来就是 $n(n-1)$ 次。但是这样就重复计算了一次，因为 $A$ 和 $B$ 握过手，算在 $A$ 握的 $(n-1)$ 次里面，$B$ 和 $A$ 握过手，算在 $B$ 握的 $n-1$ 次里面，即 $A$ 和 $B$ 一共只能握了一次手，但是在 $A$ 那边计算了一次，在 $B$ 那边也计算了一次。别的情况也都是如此，因此总共握手的次数是 $\frac{n(n-1)}{2}$
#### 1.2.2 每个点的度数：
由于每个顶点都连了其它所有的顶点，所以 $deg(v)=n-1$
#### 1.2.3 Maximally connected
- Complete Graphs是连通性最强的图

## 2. Trees
### 2.1 定义：
A tree is a connected graph that **contains no cycles（cycle最后会走回起点，且不能踩一个点两次）**.下面是四种完全等价的定义：

设图 $G=(V,E)$，顶点数 $n=∣V∣$

1. $G$ is connected and contains no cycles.
    - connected：任意两点都有路径
    - no cycles：没有回路
2. $G$ is connected and has $n−1$ edges.
3. Removing any edge disconnects G.
    - 与Complete Graphs相反，Trees是Minimally connected，拆掉任意一条边，这个图就不再连通
4. Adding any edge creates a cycle.
    - Trees是刚刚好没产生cycle，但凡加上一条边就会产生cycle

### Tips：
联想一下数据结构中学过的树，它也是 $n$ 个节点 $(n-1)$ 条边；不存在任何回路；随便删掉一条边，这棵树就没法连在一起了。所以离散数学中的 $Tree$ 其实完全符合数据结构中学过的 $Tree$

### 证明$1 \implies 2$

我们要证明 $G \ is\ connected\ \&  \ acyclic \ \implies G \ is \ connected \ \& \ |E|=|V|-1$ ，用数学归纳法：

- Base case：n=1时，这幅图只有一个顶点0条边，显然是成立的
- Inductive Hypothesis：假设任意$G=(V, E)$, $|V|=k$ 时 $|E|=k-1$ 成立
- Inductive step：取任意的图 $G=(V, \ E)\ is\ connected\ and\ acyclic$ 并且 $|V| = k+1$
    - 注意 $discussion02a$中的$Build \ Up \ Error$，我们不能直接在 $|V| = k$的 $connected$ 且 $acyclic$ 的图G中直接加一个顶点加一条边就表明这个图就是$|V| = k+1$的 $connected$ 且 $acyclic$ 的图，因为它们不一定能代表所有的$|V| = k+1$的 $connected$ 且 $acyclic$ 的图。因此我们需要从
    - Claim：$Graph\ G\  is \ connected\ and \ acyclic\ \implies\ \exists v\ \in \ V\ s.t,\ deg(v)=1$
        - 我们需要引入这个lemma（辅助定理）来完成证明，这个辅助定理很简单，其实就是这个树一定会有一个度数为1的顶点，也就是数据结构中树的叶子。
    - 我们在图G中找到一个叶子节点，并把它删除掉，把连着它的那个边也删除了，执行了这个操作之后，这棵树仍然是 $connected$ 且 $acyclic$ 的。也就是说，所有的图G执行了这个操作之后都能得到符合inductive step中 $|V|=k$ 的 $connected$ 且 $acyclic$ 的树。 
        - **所以从 $|V|=k$ 出发去构建 $|V|=k+1$的图不会出现Build Up Error**
    - 上一步中我们删除了一条边和一个点得到了图 $G'$，图 $G'$ 是符合Inductive Hypothesis的，那么图 $G'$ 把刚刚删掉的那个边和那个点加回来，就能够重新得到符合要求的所有图G。证明完毕。

### 我最初的疑惑：
我感觉证明build up error在这里不存在的那一步中，虽然证明了所有的图 $G=(V, \ E)\ is\ connected\ and\ acyclic$ 并且 $|V| = k+1$，通过减去一条边和一个点都能够得到符合Inductive Hypothesis的图 $G'$，但是通过图$G$反推出来的图$G‘$不一定能够包含世界上所有符合Inductive Hypothesis的图。

但是！！！这并不影响，我们不需要证明所有的 $G=(V, \ E)\ is\ connected\ and\ acyclic$ 并且 $|V| = k+1$都是通过所有的符合Inductive Hypothesis图 $G'=(V, \ E)\ is\ connected\ and\ acyclic$ 并且 $|V| = k$ 构建出来的。但是我们刚刚通过删掉任意一个图G中的顶点和边都能够得出一个图G’，恰恰说明了所有的图G一定能够由某一个符合Inductive Hypothesis的图 $G’$ 加一条边再加一个点而得到。这就已经能够证明了。
