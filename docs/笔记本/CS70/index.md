# CS70 离散数学和概率论

伯克利的离散数学+概率论的课程，这门课的notes写的很好，并且教授上课时将国内老师一步带过的理论讲解的非常详细，这门课如果好好学的话真的能够让自己的数学思维提高一个档次，不要只记结论而不去理解推导过程，不要仅仅为了应付去记结论，这样考完试就忘记了，对自己没有任何思维方面的帮助。

具体的理论与算法的对应关系列举如下：

- 逻辑证明：稳定匹配算法
- 图论：网络拓扑设计
- 基础数论：RSA 算法
- 多项式环：纠错码设计
- 概率论：哈希表设计、负载均衡等等
