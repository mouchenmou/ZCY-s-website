# 第四章 级数

!!! abstract
    本章研究复数项级数、幂级数、Taylor 级数和 Laurent 级数。Taylor 级数用于在解析点附近展开函数，Laurent 级数用于在奇点附近展开函数——后者是留数理论的基础。

## 4.1 复数项级数

### 4.1.1 复数列的收敛

$\{z_n = a_n + ib_n\}$ 收敛于 $z = a + ib$ 的充要条件：

$$\lim_{n \to \infty} a_n = a \quad \text{且} \quad \lim_{n \to \infty} b_n = b$$

**Cauchy 收敛准则** ：$\forall \varepsilon > 0$，$\exists N$，当 $n > N$ 时 $|z_{n+p} - z_n| < \varepsilon$（$\forall p \geq 1$）。

### 4.1.2 级数的收敛与绝对收敛

$\sum_{n=1}^{\infty} z_n$ 收敛 $\iff \sum a_n$ 和 $\sum b_n$ 都收敛。

**绝对收敛** ：$\sum |z_n|$ 收敛 $\Rightarrow \sum z_n$ 收敛。

## 4.2 幂级数

### 4.2.1 Abel 定理

**定理** ：

1. 若 $\sum c_n z^n$ 在 $z = z_0 \neq 0$ 处收敛，则对 $|z| < |z_0|$ 的所有 $z$ 都 **绝对收敛**
2. 若在 $z = z_0$ 处发散，则对 $|z| > |z_0|$ 的所有 $z$ 都 **发散**

### 4.2.2 收敛半径（Radius of Convergence）

$$R = \lim_{n \to \infty} \left|\frac{c_n}{c_{n+1}}\right| \quad \text{或} \quad R = \lim_{n \to \infty} \frac{1}{\sqrt[n]{|c_n|}}$$

- **$R > 0$** ：在 $|z| < R$ 内绝对收敛，在 $|z| > R$ 外发散
- **$R = 0$** ：仅在 $z = 0$ 处收敛
- **$R = +\infty$** ：在全平面收敛

!!! warning "收敛圆周上的行为不确定"
    在 $|z| = R$ 上，级数可能收敛也可能发散，需要 **逐点判断** 。幂级数的和函数在其收敛圆周上 **至少有一个奇点** 。

### 4.2.3 幂级数的性质

设 $\sum c_n(z-a)^n$ 的收敛半径为 $R$，和函数为 $f(z)$：

1. $f(z)$ 在收敛圆 $|z - a| < R$ 内 **解析**
2. 可 **逐项求导** ：$f'(z) = \sum n c_n(z-a)^{n-1}$
3. 可 **逐项积分** ，收敛半径不变

## 4.3 Taylor 级数

### 4.3.1 展开定理

**定理** ：若 $f(z)$ 在区域 $D$ 内解析，$z_0 \in D$，$d = \text{dist}(z_0, \partial D)$，则当 $|z - z_0| < d$ 时：

$$f(z) = \sum_{n=0}^{\infty} c_n (z - z_0)^n, \quad c_n = \frac{f^{(n)}(z_0)}{n!}$$

**收敛半径** ：$R = |z_0 - \alpha|$，其中 $\alpha$ 是距 $z_0$ 最近的奇点。

!!! example "常用 Taylor 展开"

    $$e^z = \sum_{n=0}^{\infty} \frac{z^n}{n!} \quad (R = +\infty)$$

    $$\sin z = \sum_{n=0}^{\infty} \frac{(-1)^n z^{2n+1}}{(2n+1)!} \quad (R = +\infty)$$

    $$\cos z = \sum_{n=0}^{\infty} \frac{(-1)^n z^{2n}}{(2n)!} \quad (R = +\infty)$$

    $$\frac{1}{1-z} = \sum_{n=0}^{\infty} z^n \quad (|z| < 1)$$

### 4.3.2 等价刻画

$f(z)$ 在区域 $D$ 内解析 $\iff$ 以下等价：

1. $f(z)$ 在 $D$ 内可导
2. $u, v$ 在 $D$ 内可微且满足 C-R 条件
3. $f(z)$ 在 $D$ 内连续且积分与路径无关
4. $f(z)$ 在 $D$ 内可展开为幂级数

---

## 4.4 零点孤立性

### 4.4.1 零点的定义

- **零点** ：$f(z_0) = 0$
- **孤立零点** ：$z_0$ 的某去心邻域内 $f(z) \neq 0$
- **$m$ 级零点** ：$f(z) = (z - z_0)^m \psi(z)$，其中 $\psi(z_0) \neq 0$

### 4.4.2 判定条件

$z_0$ 是 $f(z)$ 的 $m$ 级零点 $\iff$

$$f(z_0) = f'(z_0) = \cdots = f^{(m-1)}(z_0) = 0, \quad f^{(m)}(z_0) \neq 0$$

### 4.4.3 零点孤立性定理

**定理** ：不恒为零的解析函数的零点必定是 **孤立零点** 。

**推论** （唯一性定理）：若 $f(z), g(z)$ 在 $D$ 内解析，且存在一列不同点 $\{z_n\} \to z_0$ 使 $f(z_n) = g(z_n)$，则在 $D$ 内 $f \equiv g$。

## 4.5 Laurent 级数

### 4.5.1 展开定理

**定理** ：若 $f(z)$ 在圆环域 $R_1 < |z - z_0| < R_2$ 内解析，则

$$f(z) = \sum_{n=-\infty}^{\infty} c_n (z - z_0)^n$$

其中

$$c_n = \frac{1}{2\pi i} \oint_C \frac{f(\zeta)}{(\zeta - z_0)^{n+1}}d\zeta \quad (n = 0, \pm 1, \pm 2, \ldots)$$

$C$ 为圆环内绕 $z_0$ 的任意正向简单闭曲线。

Laurent 级数 = **正幂部分** （解析部分）+ **负幂部分** （主要部分）：
- 正幂部分：$\sum_{n=0}^{\infty} c_n(z-z_0)^n$
- 负幂部分：$\sum_{n=1}^{\infty} c_{-n}(z-z_0)^{-n}$

!!! warning
    Laurent 展开 **不唯一** ——同一个函数在不同圆环域内有不同的展开式。展开的关键是将各部分写成几何级数的形式。

### 4.5.2 展开技巧

利用几何级数 $\frac{1}{1-w} = \sum_{n=0}^{\infty} w^n$（$|w| < 1$）：

- **$|z - a| < R$** ：直接展开 $\frac{1}{1 - \frac{z-a}{R}}$
- **$|z - a| > R$** ：提取 $\frac{1}{z-a}$ 后展开 $\frac{1}{1 - \frac{R}{z-a}}$

!!! example "将 $f(z) = \frac{1}{(z-1)(z-2)}$ 在不同区域展开"

    先部分分式：$f(z) = \frac{1}{z-2} - \frac{1}{z-1}$

    **（1）** $0 < |z| < 1$：

    $$\frac{1}{1-z} = \sum_{n=0}^{\infty} z^n, \quad \frac{1}{2-z} = \frac{1}{2}\sum_{n=0}^{\infty}\left(\frac{z}{2}\right)^n$$

    $$f(z) = \sum_{n=0}^{\infty}\left(1 - \frac{1}{2^{n+1}}\right)z^n$$

    **（2）** $1 < |z| < 2$：

    $$\frac{1}{1-z} = -\frac{1}{z}\sum_{n=0}^{\infty}\frac{1}{z^n}, \quad \frac{1}{2-z} = \frac{1}{2}\sum_{n=0}^{\infty}\left(\frac{z}{2}\right)^n$$

    $$f(z) = -\sum_{n=0}^{\infty} z^{-(n+1)} + \sum_{n=0}^{\infty}\frac{z^n}{2^{n+1}}$$

    **（3）** $2 < |z| < +\infty$：

    $$\frac{1}{1-z} = -\frac{1}{z}\sum_{n=0}^{\infty}\frac{1}{z^n}, \quad \frac{1}{2-z} = -\frac{1}{z}\sum_{n=0}^{\infty}\frac{2^n}{z^n}$$

    $$f(z) = \sum_{n=0}^{\infty}(2^n - 1)z^{-(n+1)}$$
