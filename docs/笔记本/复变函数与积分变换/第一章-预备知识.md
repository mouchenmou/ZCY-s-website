# 第一章 预备知识 — Preliminaries

!!! abstract
    本章介绍复数的基本概念、运算规则、几何表示以及复平面上的点集拓扑。这是整个复变函数理论的基础。

## 1.1 复数及其表示

### 1.1.1 复数的定义

- 复数 $z = x + iy$，其中 $x, y \in \mathbb{R}$，$i^2 = -1$
- $x = \text{Re}(z)$ 称为实部（real part），$y = \text{Im}(z)$ 称为虚部（imaginary part）
- $z = 0 \iff x = 0$ 且 $y = 0$
- 任意两个复数 **不能比较大小**（复数域不是有序域）

### 1.1.2 共轭复数
- $\overline{z} = x - iy$
- 重要恒等式：
  $$x = \frac{z + \overline{z}}{2}, \quad y = \frac{z - \overline{z}}{2i}$$

### 1.1.3 两种表示形式

| 表示形式 | 公式                                               | 说明         |
| ---- | ------------------------------------------------ | ---------- |
| 代数形式 | $z = x + iy$                                     | 便于加减运算     |
| 指数形式 | $z = r(\cos\theta + i\sin\theta) = re^{i\theta}$ | 便于乘除、乘方、开方 |

其中：
- $r = |z| = \sqrt{x^2 + y^2}$ 称为 **模**
- $\theta = \text{Arg}\,z$ 称为 **辐角** ，有无穷多个值
- $\text{Arg}\,z = \arg z + 2k\pi$（$k \in \mathbb{Z}$），其中 $\arg z \in (-\pi, \pi]$ 称为 **主值辐角**

!!! warning "Arg z 与 arg z 的区别"
    - $\text{Arg}\,z$ 是 **集合** (无限多个值，相差 $2\pi$ 的整数倍）
    - $\arg z$ 是 **单个值** (主值，限制在 $(-\pi, \pi]$ 内）
    - 等式 $\text{Arg}(z_1 z_2) = \text{Arg}\,z_1 + \text{Arg}\,z_2$ 的意思是两边作为集合相等

$\arg z$ 的具体计算（$z \neq 0$）：

$$\arg z = \begin{cases}
\arctan\frac{y}{x}, & x > 0 \text{（第一、四象限）} \\
\frac{\pi}{2}, & x = 0, y > 0 \\
\arctan\frac{y}{x} + \pi, & x < 0, y \geq 0 \text{（第二象限）} \\
\arctan\frac{y}{x} - \pi, & x < 0, y < 0 \text{（第三象限）} \\
-\frac{\pi}{2}, & x = 0, y < 0
\end{cases}$$

### 1.1.4 几何表示

1. **点表示**：复平面上的点 $P(x, y)$
2. **向量表示**：向量 $\vec{OP} = (x, y)$

## 1.2 复数的运算

### 1.2.1 四则运算

设$z_1 = x_1 + iy_1$，$z_2 = x_2 + iy_2$：

- **加减法**：$z_1 \pm z_2 = (x_1 \pm x_2) + i(y_1 \pm y_2)$
- **乘法**：$z_1 \cdot z_2 = (x_1 x_2 - y_1 y_2) + i(x_1 y_2 + x_2 y_1)$
- **除法**（$z_2 \neq 0$）：
  $$\frac{z_1}{z_2} = \frac{x_1 x_2 + y_1 y_2}{x_2^2 + y_2^2} + i\frac{x_2 y_1 - x_1 y_2}{x_2^2 + y_2^2}$$

!!! example "除法为什么乘以共轭？"
    $$\frac{z_1}{z_2} = \frac{z_1 \overline{z_2}}{z_2 \overline{z_2}} = \frac{z_1 \overline{z_2}}{|z_2|^2}$$
    分母变为实数，分子展开即可。

### 1.2.2 三角不等式

$$|z_1 + z_2| \leq |z_1| + |z_2|$$
$$|z_1 - z_2| \geq \big||z_1| - |z_2|\big|$$

几何意义：三角形两边之和大于第三边。

### 1.2.3 乘除法的几何意义

设 $z_1 = r_1 e^{i\theta_1}$，$z_2 = r_2 e^{i\theta_2}$：

$$|z_1 z_2| = |z_1| \cdot |z_2|, \quad \text{Arg}(z_1 z_2) = \text{Arg}\,z_1 + \text{Arg}\,z_2$$
$$\left|\frac{z_1}{z_2}\right| = \frac{|z_1|}{|z_2|}, \quad \text{Arg}\left(\frac{z_1}{z_2}\right) = \text{Arg}\,z_1 - \text{Arg}\,z_2$$

几何上，$z_1 z_2$ 相当于将 $z_2$ 的模扩大 $|z_1|$ 倍并旋转角度 $\text{Arg}\,z_1$。

### 1.2.4 乘方与开方

**乘方**：
$$z^n = r^n e^{in\theta} = r^n(\cos n\theta + i\sin n\theta)$$

特别地，当 $r = 1$ 时：
$$(\cos\theta + i\sin\theta)^n = \cos n\theta + i\sin n\theta \quad \text{(De Moivre 公式)}$$

!!! proof "证明 Euler 公式 $e^{i\theta} = \cos\theta + i\sin\theta$"
    将 $e^x$、$\cos x$、$\sin x$ 的 Maclaurin 级数写出：
    $$e^x = \sum_{n=0}^{\infty}\frac{x^n}{n!}, \quad \cos x = \sum_{n=0}^{\infty}\frac{(-1)^n x^{2n}}{(2n)!}, \quad \sin x = \sum_{n=0}^{\infty}\frac{(-1)^n x^{2n+1}}{(2n+1)!}$$
    将 $x = i\theta$ 代入 $e^x$ 的展开式：
    $$e^{i\theta} = 1 + i\theta + \frac{(i\theta)^2}{2!} + \frac{(i\theta)^3}{3!} + \frac{(i\theta)^4}{4!} + \frac{(i\theta)^5}{5!} + \cdots$$
    利用 $i^2 = -1,\; i^3 = -i,\; i^4 = 1,\; \ldots$ 分离实部与虚部：
    $$e^{i\theta} = \left(1 - \frac{\theta^2}{2!} + \frac{\theta^4}{4!} - \cdots\right) + i\left(\theta - \frac{\theta^3}{3!} + \frac{\theta^5}{5!} - \cdots\right)$$
    括号内分别是 $\cos\theta$ 和 $\sin\theta$ 的级数展开，故：
    $$e^{i\theta} = \cos\theta + i\sin\theta$$

!!! proof "证明 De Moivre 公式 $(\cos\theta + i\sin\theta)^n = \cos n\theta + i\sin n\theta$"
    由 Euler 公式 $(\cos\theta + i\sin\theta) = e^{i\theta}$，两边取 $n$ 次方：
    $$(\cos\theta + i\sin\theta)^n = (e^{i\theta})^n = e^{in\theta} = \cos n\theta + i\sin n\theta$$
    证毕。

**开方**：若 $w^n = z$，则 $w = \sqrt[n]{z}$ 有 $n$ 个值：

$$w_k = \sqrt[n]{|z|} \cdot e^{i\frac{\arg z + 2k\pi}{n}} = \sqrt[n]{|z|}\left(\cos\frac{\arg z + 2k\pi}{n} + i\sin\frac{\arg z + 2k\pi}{n}\right)$$

其中 $k = 0, 1, 2, \ldots, n-1$。

!!! example "开方的几何解释"
    $n$ 个根均匀分布在以原点为圆心、$\sqrt[n]{|z|}$ 为半径的圆上，构成正 $n$ 边形的顶点。

## 1.3 复球面与无穷远点

- 通过 **球极投影** ，复平面上的点与球面上的点（除北极点 $N$ 外）一一对应
- 北极点 $N$ 对应 **无穷远点** $\infty$
- 加上 $\infty$ 的复平面称为 **扩充复平面** ，记为 $\overline{\mathbb{C}} = \mathbb{C} \cup \{\infty\}$

**约定运算**（$a \neq 0, a \neq \infty$）：

| 运算 | 结果 |
|------|------|
| $a \pm \infty = \infty \pm a$ | $\infty$ |
| $a \cdot \infty = \infty \cdot a$ | $\infty$ |
| $\frac{a}{\infty} = 0$ | $0$ |
| $\frac{\infty}{a} = \infty$ | $\infty$ |

!!! warning
    以下运算**无定义**：$\infty \pm \infty$、$0 \cdot \infty$、$\frac{\infty}{\infty}$、$\frac{0}{0}$

## 1.4 复平面上的点集 — Point Sets in the Complex Plane

### 1.4.1 邻域（Neighborhood）

- $z_0$ 的 $\delta$ 邻域：$D(z_0, \delta) = \{z : |z - z_0| < \delta\}$
- $z_0$ 的**去心邻域**（deleted neighborhood）：$\{z : 0 < |z - z_0| < \delta\}$
- 无穷远点的邻域：$\{z : |z| > M\}$（包含 $\infty$ 本身）
- 无穷远点的去心邻域：$\{z : M < |z| < +\infty\}$

### 1.4.2 内点、开集、边界

| 概念 | 定义 |
|------|------|
| 内点（Interior point） | $z_0 \in E$，存在邻域全含于 $E$ |
| 开集（Open set） | $E$ 的每个点都是内点 |
| 边界点（Boundary point） | $z_0$ 的任意邻域内既有属于 $E$ 的点也有不属于 $E$ 的点 |
| 边界 $\partial E$ | 所有边界点的全体 |

### 1.4.3 区域（Region/Domain）

区域 = **连通的开集**

- 连通（connected）：$E$ 中任意两点可用完全属于 $E$ 的折线连接
- 闭区域 $\overline{D} = D \cup \partial D$（区域 + 边界）
- 有界区域：存在 $M > 0$ 使 $\forall z \in D, |z| \leq M$

### 1.4.4 曲线与连通性

| 曲线类型  | 条件               |
| ----- | ---------------- |
| 简单曲线  | 没有重点（自交点）        |
| 简单闭曲线 | 简单曲线且起点=终点       |
| 单连通域  | 任意简单闭曲线的内部都属于该区域 |
| 多连通域  | 不是单连通域           |

### 1.4.5 平面图形的复数表示

由于平面图形上的点可以用复数表示，很多平面图形能用复数形式的方程（或不等式）来描述；反之，也可以通过给定的复数方程（或不等式）来确定它所表示的平面图形。


!!! example "例3 — 直线的复数方程"
    **题目**：将通过两点 $z_1 = x_1 + iy_1$ 与 $z_2 = x_2 + iy_2$ 的直线用复数方程来表示。

    **解**：通过点 $(x_1, y_1)$ 与 $(x_2, y_2)$ 的直线可用参数方程表示为
    $$\begin{cases} x = x_1 + t(x_2 - x_1) \\ y = y_1 + t(y_2 - y_1) \end{cases} \quad (-\infty < t < +\infty)$$
    因此，它的复数形式的参数方程为
    $$z = z_1 + t(z_2 - z_1) \quad (-\infty < t < +\infty)$$
    由此得知由 $z_1$ 到 $z_2$ 的线段的参数方程为
    $$z = z_1 + t(z_2 - z_1) \quad (0 \leq t \leq 1)$$
    取 $t = \frac{1}{2}$，得知线段 $z_1 z_2$ 的中点为
    $$z = \frac{z_1 + z_2}{2}$$

!!! example "例4 — 求方程所表示的曲线"
    **题目**：求下列方程所表示的曲线：
    
    1. $|z + i| = 2$
    2. $|z - 2i| = |z + 2|$
    3. $\text{Im}(z + i) = 4$

    **解**：

    **(1)** $|z + i| = 2$

    设 $z = x + iy$，方程变为
    $$|x + (y + 1)i| = 2 \implies x^2 + (y + 1)^2 = 4$$
    这是以 $(0, -1)$ 为圆心、半径为 2 的圆。

    
    **(2)** $|z - 2i| = |z + 2|$

    几何上，该方程表示到点 $2i$ 和 $-2$ 的距离相等的点的轨迹，所以它表示连接 $2i$ 和 $-2$ 的线段的**垂直平分线**，方程为 $y = -x$。

    也可以代数验证：设 $z = x + iy$，
    $$x^2 + (y - 2)^2 = (x + 2)^2 + y^2$$
    $$x^2 + y^2 - 4y + 4 = x^2 + 4x + 4 + y^2$$
    $$-4y = 4x \implies y = -x$$

    ![](附件/Pasted%20image%2020260515223343.png)

    **(3)** $\text{Im}(\overline{z} + i) = 4$

    设 $z = x + iy$，则 $\overline{z} + i = (x - iy) + i = x + (1 - y)i$，
    $$\text{Im}(\overline{z} + i) = 1 - y = 4 \implies y = -3$$
    所求曲线为水平直线 $y = -3$。

    ![](附件/Pasted%20image%2020260515223552.png)

!!! example "例5 — 求不等式所表示的几何图形"
    **题目**：求下列不等式所表示的几何图形：
    1. $|z - \alpha| < R$
    2. $\alpha < \arg z < \beta \quad (-\pi < \alpha < \beta < \pi)$
    3. $0 \leq \text{Im}(z) < 2\pi$

    **解**：

    **(1)** $|z - \alpha| < R$

    满足不等式的点 $z$ 的全体是以 $\alpha$ 为中心、$R$ 为半径的**圆内部（开圆盘）**，是一个**有界区域**。

    ![](附件/Pasted%20image%2020260515223239.png)

    **(2)** $\alpha < \arg z < \beta$

    满足不等式的点 $z$ 的全体是以原点 $O$ 为顶点、张角从 $\alpha$ 到 $\beta$ 的**角形区域**，是一个**无界区域**。

    ![](附件/Pasted%20image%2020260515223722.png)

    **(3)** $0 \leq \text{Im}(z) < 2\pi$

    满足不等式的点 $z$ 的全体是 $z$ 平面上介于 $y = 0$ 与 $y = 2\pi$ 之间的一条**水平带状区域**（包含下边界 $y = 0$，不包含上边界 $y = 2\pi$），是一个**无界区域**。
