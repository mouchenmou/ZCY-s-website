# 第二章 解析函数

!!! abstract
    本章研究复变函数的极限、连续、可导与解析性。核心内容是 Cauchy-Riemann 方程——判断函数解析的充要条件，以及解析函数与调和函数的关系。

## 2.1 复变函数

### 2.1.1 定义

- 复变函数 $w = f(z)$，其中 $z \in D \subseteq \mathbb{C}$
- 分解为实部和虚部：$f(z) = f(x + iy) = u(x, y) + iv(x, y)$
- 一个复变函数本质上对应 **两个二元实函数** $u$ 和 $v$

### 2.1.2 极限

$$\lim_{z \to z_0} f(z) = A \iff \forall \varepsilon > 0,\; \exists \delta > 0,\; 0 < |z - z_0| < \delta \Rightarrow |f(z) - A| < \varepsilon$$

**等价定义**：$f(z) = u + iv$，$A = u_0 + iv_0$，$z_0 = x_0 + iy_0$

$$\lim_{z \to z_0} f(z) = A \iff \begin{cases} \lim\limits_{\substack{x \to x_0 \\ y \to y_0}} u(x,y) = u_0 \\ \lim\limits_{\substack{x \to x_0 \\ y \to y_0}} v(x,y) = v_0 \end{cases}$$

!!! warning "复极限比实极限更严格"
    复平面上 $z \to z_0$ 可以从==任意方向==逼近，因此极限存在的条件比实函数更强。如果沿两条不同路径得到的极限值不同，则极限不存在。

!!! example "极限不存在的证明"
    证明 $f(z) = \frac{\text{Re}(z)}{|z|}$ 当 $z \to 0$ 时极限不存在。
    
    设 $z = x + iy$，则 $f(z) = \frac{x}{\sqrt{x^2 + y^2}}$。
    
    - 沿 $x$ 轴逼近（$y = 0$）：极限为 $\frac{x}{|x|} = \pm 1$
    - 沿 $y$ 轴逼近（$x = 0$）：极限为 $0$
    
    不同路径极限不同，故极限不存在。

### 2.1.3 连续性

$$f(z) \text{ 在 } z_0 \text{ 连续} \iff \lim_{z \to z_0} f(z) = f(z_0)$$

性质与实连续函数类似：四则运算、复合、有界闭区域上的最值定理和一致连续性都成立。

## 2.2 解析函数

### 2.2.1 可导与解析的定义

#### **可导的定义**：
如果 $f'(z_0) = \lim_{\Delta z \to 0} \frac{f(z_0 + \Delta z) - f(z_0)}{\Delta z}$ 存在。那么 $f(z)$ 在 $z_0$ 可导。

注意：$\Delta z \to 0$ 的方式是任意的。

#### **解析的定义**：

- $f(z)$ 在 $z_0$ 解析 $\iff$ $f(z)$ 在 $z_0$ 的==某个邻域内==处处可导
- $f(z)$ 在区域 $D$ 内解析 $\iff$ $f(z)$ 在 $D$ 内==处处可导==
- 在全平面解析的函数称为==整函数==

!!! warning "可导 ≠ 解析"
    - 在一点：解析 $\Rightarrow$ 可导，但可导 $\nRightarrow$ 解析
    - 在区域内：解析 $\iff$ 可导
    - 解析函数不可能仅在一个点或一条曲线上解析；所有解析点的集合必为开集

!!! question "为什么只要一阶可导就无穷次可导？"
    这是复变函数与实变函数的本质区别。实函数可以一阶可导但二阶不可导（如 $f(x) = x^{4/3}$ 在 $x = 0$）。而复变函数只要在区域内可导，就能推出无穷次可导——这是后面 Cauchy 积分公式和高阶导数公式的推论。

### 2.2.2 Cauchy-Riemann 方程（C-R 方程）

**定理**：$f(z) = u(x,y) + iv(x,y)$ 在区域 $D$ 内解析的 **充要条件** ：

1. $u, v$ 在 $D$ 内可微
2. 满足 **C-R 方程**：$\frac{\partial u}{\partial x} = \frac{\partial v}{\partial y}, \quad \frac{\partial v}{\partial x} = -\frac{\partial u}{\partial y}$

!!! explanation "为什么 C-R 方程是解析的充要条件？"
    **必要性（$\Rightarrow$）**：假设 $f'(z_0)$ 存在。
    
    $\Delta z \to 0$ 的方式是任意的，我们取两条特殊路径：
    
    - **沿实轴**逼近（$\Delta y = 0$，$\Delta z = \Delta x$）：
      $、$f'(z_0) = \lim_{\Delta x \to 0}\frac{u(x+\Delta x,y) - u(x,y) + i[v(x+\Delta x,y) - v(x,y)]}{\Delta x} = u_x + i v_x$
    - **沿虚轴**逼近（$\Delta x = 0$，$\Delta z = i\Delta y$）：
      $f'(z_0) = \lim_{\Delta y \to 0}\frac{u(x,y+\Delta y) - u(x,y) + i[v(x,y+\Delta y) - v(x,y)]}{i\Delta y} = \frac{u_y + i v_y}{i} = v_y - i u_y$
    
    由于导数唯一，两式实部与虚部分别相等：$u_x = v_y, \quad v_x = -u_y$
    
    这就是 C-R 方程。

    ---

    **充分性（$\Leftarrow$）**：假设 $u, v$ 可微且满足 C-R 方程。
    
    由可微性，增量可写为：$\Delta u = u_x \Delta x + u_y \Delta y + o(|\Delta z|), \quad \Delta v = v_x \Delta x + v_y \Delta y + o(|\Delta z|)$
    
    $\Delta f = \Delta u + i \Delta v = (u_x + i v_x)\Delta x + (u_y + i v_y)\Delta y + o(|\Delta z|)$
    
    代入 C-R 方程 $u_y = -v_x,\; v_y = u_x$：
    
    $$\Delta f = (u_x + i v_x)\Delta x + (-v_x + i u_x)\Delta y + o(|\Delta z|)$$
    
    $$= (u_x + i v_x)(\Delta x + i\Delta y) + o(|\Delta z|)$$
    
    $$= (u_x + i v_x)\Delta z + o(|\Delta z|)$$
    
    所以：
    
    $$f'(z_0) = \lim_{\Delta z \to 0}\frac{\Delta f}{\Delta z} = u_x + i v_x$$
    
    极限存在且与路径无关，故 $f(z)$ 可导。

**一点可导的充要条件**：偏导数 $u_x, u_y, v_x, v_y$ 在该点存在且满足 C-R 方程。

此时导数为：
$f'(z) = \frac{\partial u}{\partial x} + i\frac{\partial v}{\partial x} = \frac{\partial v}{\partial y} - i\frac{\partial u}{\partial y}$

!!! example "判断 $f(z) = |z|^2$ 的可导性与解析性"
    $f(z) = x^2 + y^2$，即 $u = x^2 + y^2$，$v = 0$。
    
    C-R 方程要求：
    
    - $u_x = v_y \Rightarrow 2x = 0 \Rightarrow x = 0$
    - $v_x = -u_y \Rightarrow 0 = -2y \Rightarrow y = 0$
    
    仅在 $z = 0$ 处满足 C-R 方程，故 $f(z)$ 仅在 $z = 0$ 可导。
    
    但 $z = 0$ 的 **任何邻域** 内都有不满足 C-R 方程的点，所以 $f(z)$ **处处不解析**。

### 2.2.3 Wirtinger 偏导数

#### 偏导数的定义：

$$
\frac{\partial}{\partial z} = \frac{1}{2}\left(\frac{\partial}{\partial x} - i\frac{\partial}{\partial y}\right), \quad \frac{\partial}{\partial \overline{z}} = \frac{1}{2}\left(\frac{\partial}{\partial x} + i\frac{\partial}{\partial y}\right)
$$

**重要结论**：$f(z)$ 解析 $\iff \frac{\partial f}{\partial \overline{z}} \equiv 0$

即解析函数不依赖于 $\overline{z}$，只依赖于 $z$。

!!! success "证明 $\frac{\partial f}{\partial \overline{z}} = 0$"
    $$\frac{\partial f}{\partial \overline{z}} = \frac{1}{2}\left(\frac{\partial u}{\partial x} + i\frac{\partial v}{\partial x} + i\frac{\partial u}{\partial y} - \frac{\partial v}{\partial y}\right)$$
    $$= \frac{1}{2}\left(\frac{\partial u}{\partial x} - \frac{\partial v}{\partial y}\right) + \frac{i}{2}\left(\frac{\partial v}{\partial x} + \frac{\partial u}{\partial y}\right)$$
    由 C-R 方程，两项均为 0。

### 2.2.4 解析函数退化为常数的条件

以下任一条件成立时，解析函数 $f(z)$ 必为常数：

| 条件                    | 结论      |     |     |
| --------------------- | ------- | --- | --- |
| $f'(z) \equiv 0$      | $f$ 为常数 |     |     |
| $\text{Re}(f)$ 为常数    | $f$ 为常数 |     |     |
| $\text{Im}(f)$ 为常数    | $f$ 为常数 |     |     |
| $\|f(z)\|$ 为常数        | $f$ 为常数 |     |     |
| $\arg f(z)$ 为常数       | $f$ 为常数 |     |     |
| $\overline{f(z)}$ 也解析 | $f$ 为常数 |     |     |

## 2.3 调和函数

### 2.3.1 定义

如果满足下面两个性质，那么 $u(x,y)$ 是区域 $D$ 内的调和函数：

1. $u$ 有 **二阶连续偏导数**
2. 满足 **Laplace 方程**：$\Delta u = u_{xx} + u_{yy} = 0$

### 2.3.2 解析函数与调和函数的关系

**定理**：若 $f(z) = u + iv$ 解析，则 $u$ 和 $v$ 都是调和函数。

**共轭调和函数**：若 $u, v$ 都是调和函数且满足 C-R 方程，则称 $v$ 是 $u$ 的共轭调和函数。

$$f(z) \text{ 解析} \iff v \text{ 是 } u \text{ 的共轭调和函数}$$

!!! warning
    调和函数的共轭不唯一，相差一个常数。给定 $u$ 后，$v$ 由 C-R 方程确定到一个常数。

### 2.3.3 由已知调和函数求解析函数

已知调和函数 $u(x,y)$，求解析函数 $f(z) = u + iv$，有三种方法：

| 方法        | 思路                                                 | 适用场景 |
| --------- | -------------------------------------------------- | ---- |
| 法一：C-R 积分 | 由 $v_y = u_x$ 对 $y$ 积分得 $v$，再由 $v_x = -u_y$ 确定积分常数 | 通用   |
| 法二：全微分    | $dv = -u_y dx + u_x dy$，沿路径积分                      | 通用   |
| 法三：复变量    | $f'(z) = u_x - iu_y$，将 $x+iy$ 换为 $z$ 后积分           | 最快捷  |

!!! example "已知 $u = x^2 - y^2 + xy$，$f(0) = 0$，求 $f(z)$"

    **法三（复变量法）**：
    
    $$
    f'(z) = u_x - iu_y = (2x + y) - i(-2y + x) = (2-i)x + (1+2i)y = (2-i)(x + iy) = (2-i)z
    $$
    
    
    $$
    f(z) = \int (2-i)z \, dz = \left(1 - \frac{i}{2}\right)z^2 + C
    $$
    
    
    由 $f(0) = 0$ 得 $C = 0$，
    
    故 $f(z) = \left(1 - \frac{i}{2}\right)z^2$。

## 2.4 初等解析函数
### 2.4.1 指数函数 $e^z$

$$e^z = e^{x+iy} = e^x(\cos y + i\sin y)$$

| 性质 | 说明 |
|------|------|
| 全平面解析 | $(e^z)' = e^z$ |
| 永不为零 | $e^z \neq 0$（因 $e^x > 0$，$e^{iy} \neq 0$） |
| 加法定理 | $e^{z_1} \cdot e^{z_2} = e^{z_1 + z_2}$ |
| 周期性 | 基本周期为 $2\pi i$：$e^{z + 2k\pi i} = e^z$ |
| 无穷远极限 | $\lim\limits_{z \to \infty} e^z$ 不存在（$x \to +\infty$ 时 $\to \infty$，$x \to -\infty$ 时 $\to 0$） |

!!! warning
    虽然 $e^z = e^{z + 2k\pi i}$，但 $e^z \neq 0$。这说明 **Rolle 中值定理在复平面不成立**；但 L'Hospital 法则仍然成立。

### 2.4.2 三角函数

$$\sin z = \frac{e^{iz} - e^{-iz}}{2i}, \quad \cos z = \frac{e^{iz} + e^{-iz}}{2}$$

1. **Euler 公式仍然成立**：$\sin z = \frac{e^{iz} - e^{-iz}}{2i}$，$\cos z = \frac{e^{iz} + e^{-iz}}{2}$
2. **在整个复平面解析**：$(\sin z)' = \cos z$，$(\cos z)' = -\sin z$
3. **三角恒等式仍然成立**：$\sin(z_1 \pm z_2) = \sin z_1 \cos z_2 \pm \cos z_1 \sin z_2$，$\cos(z_1 \pm z_2) = \cos z_1 \cos z_2 \mp \sin z_1 \sin z_2$，$\sin^2 z + \cos^2 z = 1$ 等（半角公式除外）
4. **奇偶性**：$\sin z$ 为奇函数，$\cos z$ 为偶函数
5. **周期性**：基本周期为 $2\pi$
6. **无界性**：$|\sin z|$ 和 $|\cos z|$ 可以无界，这与实三角函数完全不同。例如 $|\cos i| = \frac{e^{-1} + e}{2} > 1$，$|\cos(iy)| \to \infty$（$y \to \infty$）
7. **其他三角函数定义**：$\tan z = \frac{\sin z}{\cos z}$，$\cot z = \frac{\cos z}{\sin z}$，$\sec z = \frac{1}{\cos z}$，$\csc z = \frac{1}{\sin z}$

!!! warning
    复三角函数==无界==，这与实三角函数完全不同！

### 2.4.3 双曲函数

$$\text{sh}\,z = \frac{e^z - e^{-z}}{2}, \quad \text{ch}\,z = \frac{e^z + e^{-z}}{2}$$

与三角函数的关系：$\text{sh}(iz) = i\sin z$，$\text{ch}(iz) = \cos z$

### 2.4.4 对数函数（多值函数）

$$\text{Ln}\,z = \ln|z| + i\text{Arg}\,z = \ln|z| + i(\arg z + 2k\pi), \quad k \in \mathbb{Z}$$

**主值支**：$\ln z = \ln|z| + i\arg z$

| 性质   | 说明                                                           |
| ---- | ------------------------------------------------------------ |
| 多值性  | 无穷多值，每两个值相差 $2\pi i$ 的整数倍                                    |
| 定义域  | $\{z : 0 < \lvert z \rvert < +\infty\}$                      |
| 乘法公式 | $\text{Ln}(z_1 z_2) = \text{Ln}\,z_1 + \text{Ln}\,z_2$（集合相等） |
| 解析性  | 除去原点和负实轴，$\ln z$ 处处解析，$(\ln z)' = \frac{1}{z}$               |

!!! warning "主值支与多值函数的区别"
    - $\ln(z_1 z_2) = \ln z_1 + \ln z_2$ **不成立**（主值支可能不满足）
    - $\text{Ln}(z_1 z_2) = \text{Ln}\,z_1 + \text{Ln}\,z_2$ **成立**（作为集合）

### 2.4.5 幂函数（多值函数）

$$z^\alpha = e^{\alpha\,\text{Ln}\,z}$$

| $\alpha$ 类型 | 值的个数 | 例子 |
|-------------|---------|------|
| $\alpha = n \in \mathbb{Z}$ | 单值 | $z^2$ |
| $\alpha = \frac{1}{n}$ | $n$ 值 | $\sqrt[n]{z}$ |
| $\alpha = \frac{m}{n}$（既约） | $n$ 值 | $z^{3/4}$ |
| $\alpha$ 无理数或 $\text{Im}(\alpha) \neq 0$ | 无穷多值 | $z^i$ |
