# 第五章 留数

!!! abstract
    本章研究孤立奇点的分类与留数计算。留数定理将复积分的计算转化为求留数的代数和，并可用于计算多种类型的实积分——这是复变函数最强大的应用之一。

## 5.1 孤立奇点的分类

### 5.1.1 定义

$z_0$ 是 $f(z)$ 的 **孤立奇点** ，如果 $f(z)$ 在 $z_0$ 不解析，但在 $z_0$ 的某个去心邻域 $0 < |z - z_0| < R$ 内处处解析。

### 5.1.2 三类孤立奇点

- **可去奇点** （Removable）：Laurent 展开无负幂项；$\lim\limits_{z \to z_0} f(z)$ 存在且有限；等价于 $f$ 在该邻域内有界
- **极点** （Pole）：Laurent 展开有有限个负幂项；$\lim\limits_{z \to z_0} f(z) = \infty$；等价于 $f(z) = \frac{\psi(z)}{(z-z_0)^m}$，$\psi(z_0) \neq 0$
- **本性奇点** （Essential）：Laurent 展开有无穷多个负幂项；$\lim\limits_{z \to z_0} f(z)$ 不存在

### 5.1.3 极点的判定

$z_0$ 是 $f(z)$ 的 **$m$ 级极点** $\iff f(z) = \frac{\psi(z)}{(z-z_0)^m}$，$\psi(z)$ 在 $z_0$ 解析且 $\psi(z_0) \neq 0$。

**与零点的关系** ：$z_0$ 是 $f(z)$ 的 $m$ 级极点 $\iff z_0$ 是 $\frac{1}{f(z)}$ 的 $m$ 级零点。

### 5.1.4 亚纯函数

在复平面上 **除极点外无其他类型奇点** 的函数称为亚纯函数。有理函数都是亚纯函数。

!!! example "判断奇点类型"
    - $f(z) = \frac{\sin z}{z}$：$z = 0$ 是 **可去奇点** （$\lim\limits_{z \to 0}\frac{\sin z}{z} = 1$）
    - $f(z) = \frac{1}{z^2 + 1}$：$z = \pm i$ 是 **一级极点**
    - $f(z) = e^{1/z}$：$z = 0$ 是 **本性奇点** （Laurent 展开有无穷多负幂项）

## 5.2 留数定理

### 5.2.1 留数的定义

$f(z)$ 在孤立奇点 $z_0$ 处的 **留数** 是其 Laurent 展开中 $(z - z_0)^{-1}$ 项的系数 $c_{-1}$，记为 $\text{Res}[f(z), z_0]$。

### 5.2.2 留数定理

**定理** ：若 $f(z)$ 在区域 $D$ 内除有限个孤立奇点 $z_1, z_2, \ldots, z_n$ 外处处解析，$C$ 为包围这些奇点的正向简单闭曲线，则

$$\oint_C f(z)\,dz = 2\pi i \sum_{k=1}^{n} \text{Res}[f(z), z_k]$$

!!! question "留数定理的本质"
    把沿闭曲线的积分转化为代数运算——只需找到被积函数的所有奇点并计算留数。

### 5.2.3 留数的计算规则

**规则 1（一级极点）** ：

$$\text{Res}[f(z), z_0] = \lim_{z \to z_0} (z - z_0)f(z)$$

**规则 2（$m$ 级极点）** ：

$$\text{Res}[f(z), z_0] = \frac{1}{(m-1)!} \lim_{z \to z_0} \frac{d^{m-1}}{dz^{m-1}}\left[(z - z_0)^m f(z)\right]$$

**规则 3（$\frac{P(z)}{Q(z)}$ 型一级极点）** ：若 $P(z_0) \neq 0$，$Q(z_0) = 0$，$Q'(z_0) \neq 0$，则

$$\text{Res}\left[\frac{P(z)}{Q(z)}, z_0\right] = \frac{P(z_0)}{Q'(z_0)}$$

**规则 4** ：若 $z_0$ 是 $g(z)$ 的 $k$ 级零点，是 $h(z)$ 的 $k+1$ 级零点，则 $z_0$ 是 $f(z) = \frac{g(z)}{h(z)}$ 的单极点，且

$$\text{Res}[f(z), z_0] = (k+1)\frac{g^{(k)}(z_0)}{h^{(k+1)}(z_0)}$$

**规则 5（二级极点特化）** ：若 $g(z_0) \neq 0$，$h(z_0) = h'(z_0) = 0$，$h''(z_0) \neq 0$，则

$$\text{Res}\left[\frac{g(z)}{h(z)}, z_0\right] = \frac{2g'(z_0)}{h''(z_0)} - \frac{2}{3}\frac{g(z_0)h'''(z_0)}{[h''(z_0)]^2}$$

!!! example
    计算 $\text{Res}\left[\frac{e^z}{z^2 + 1}, i\right]$。

    $Q(z) = z^2 + 1$，$Q'(z) = 2z$，$Q'(i) = 2i \neq 0$，用规则 3：

    $$\text{Res} = \frac{P(i)}{Q'(i)} = \frac{e^i}{2i}$$

## 5.3 无穷远点的留数

### 5.3.1 定义

$$\text{Res}[f(z), \infty] = \frac{1}{2\pi i} \oint_{C^-} f(z)\,dz = -c_{-1}$$

其中 $C^-$ 为绕原点的顺时针闭曲线，$c_{-1}$ 是 $f(z)$ 在 $R < |z| < \infty$ 的 Laurent 展开中 $z^{-1}$ 的系数。

!!! warning "注意符号"
    无穷远点的留数等于 $z^{-1}$ 系数的 **相反数** （因为方向取顺时针）。

### 5.3.2 全平面留数总和定理

若 $f(z)$ 在扩充复平面上只有有限个孤立奇点，则

$$\sum_{k=1}^{n} \text{Res}[f(z), z_k] + \text{Res}[f(z), \infty] = 0$$

### 5.3.3 规则 6（利用无穷远点计算留数）

$$\text{Res}[f(z), \infty] = -\text{Res}\left[f\left(\frac{1}{z}\right) \cdot \frac{1}{z^2}, 0\right]$$

当内部极点太多、计算复杂时，用无穷远点的留数往往更简便。

!!! example "计算 $\oint_{|z|=4} \frac{z^{15}}{(z^4+2)^3(z^2+1)^2}dz$"

    内部有 6 个极点（计算繁琐），改用无穷远点：

    $$\text{Res}[f(z), \infty] = -\text{Res}\left[\frac{1}{z(1+2z^4)^3(1+z^2)^2}, 0\right]$$

    $$= -\lim_{z \to 0} z \cdot \frac{1}{z(1+2z^4)^3(1+z^2)^2} = -1$$

    $$\oint_{|z|=4} f(z)dz = -2\pi i \cdot \text{Res}[f, \infty] = 2\pi i$$

## 5.4 留数定理的应用

### 5.4.1 类型 I：三角函数积分

$$\int_0^{2\pi} R(\cos\theta, \sin\theta)\,d\theta$$

令 $z = e^{i\theta}$，则 $\cos\theta = \frac{z + z^{-1}}{2}$，$\sin\theta = \frac{z - z^{-1}}{2i}$，$d\theta = \frac{dz}{iz}$：

$$I = \oint_{|z|=1} R\left(\frac{z+z^{-1}}{2}, \frac{z-z^{-1}}{2i}\right)\frac{dz}{iz}$$

### 5.4.2 类型 II：有理函数积分

$$\int_{-\infty}^{+\infty} f(x)\,dx = 2\pi i \sum \text{Res}[f(z), \text{上半平面极点}]$$

条件：$f(z)$ 在实轴上无极点，且 $|f(z)| \leq \frac{M}{|z|^p}$（$p > 1$）。

### 5.4.3 类型 III：Fourier 型积分

$$\int_{-\infty}^{+\infty} e^{i\alpha x} f(x)\,dx = 2\pi i \sum \text{Res}[f(z)e^{i\alpha z}, \text{上半平面极点}] \quad (\alpha > 0)$$

由此可得：

$$\int_{-\infty}^{+\infty} \cos(\alpha x)f(x)\,dx = \text{Re}(I), \quad \int_{-\infty}^{+\infty} \sin(\alpha x)f(x)\,dx = \text{Im}(I)$$

!!! example "计算 $\int_0^{+\infty} \frac{x\sin x}{x^2 + a^2}dx$（$a > 0$）"

    $$\int_{-\infty}^{+\infty} \frac{x}{x^2 + a^2}e^{ix}dx = 2\pi i \cdot \text{Res}\left[\frac{ze^{iz}}{z^2 + a^2}, ai\right]$$

    $$= 2\pi i \lim_{z \to ai}\frac{ze^{iz}}{z + ia} = 2\pi i \cdot \frac{ai \cdot e^{-a}}{2ai} = \pi i e^{-a}$$

    $$\int_0^{+\infty} \frac{x\sin x}{x^2 + a^2}dx = \frac{1}{2}\text{Im}(\pi i e^{-a}) = \frac{\pi}{2}e^{-a}$$
