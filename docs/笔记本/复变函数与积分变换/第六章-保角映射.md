# 第六章 保角映射 — Conformal Mapping

!!! abstract
    本章研究复变函数的几何性质——保角映射。分式线性映射是核心内容，它具有保角性、保圆性和保对称性。初等函数（幂函数、指数函数）的映射也有重要应用。

## 6.1 保角映射的概念 — Concept of Conformal Mapping

### 6.1.1 定义

- **保角性**：映射 $w = f(z)$ 在 $z_0$ 处保持两条曲线之间的夹角（大小和方向）不变
- **保角映射/共形映射**：$w = f(z)$ 在 $z_0$ 的邻域内是**一一的**（双射）且具有保角性
- **局部保角映射**：在 $z_0$ 的某邻域内保角

### 6.1.2 保角的条件

**定理**：若 $f(z)$ 在 $z_0$ 解析且 $f'(z_0) \neq 0$，则 $w = f(z)$ 在 $z_0$ 处保角。

**几何意义**：$f'(z_0) = |f'(z_0)|e^{i\arg f'(z_0)}$
- 模 $|f'(z_0)|$ 表示**伸缩率**
- 辐角 $\arg f'(z_0)$ 表示**旋转角**

## 6.2 分式线性映射 — Bilinear (Möbius) Transformations

### 6.2.1 定义

$$w = \frac{az + b}{cz + d}, \quad ad - bc \neq 0$$

**逆映射**：$z = \frac{-dw + b}{cw - a}$

**分解**：任何分式线性映射由三种基本映射复合而成：

| 基本映射 | 公式 | 几何意义 |
|---------|------|---------|
| 平移 | $w = z + b$ | 沿向量 $b$ 平移 |
| 旋转与伸缩 | $w = az$（$a = \lambda e^{i\alpha}$） | 旋转 $\alpha$ 角，伸缩 $\lambda$ 倍 |
| 反演 | $w = \frac{1}{z}$ | 关于单位圆对称 + 关于实轴对称 |

### 6.2.2 三大几何性质

| 性质 | 内容 |
|------|------|
| **保角性** | 在扩充复平面上一一对应且具有保角性 |
| **保圆性** | 将圆周（含直线，视为无穷大半径的圆）映射为圆周 |
| **保对称性** | 关于圆周的对称点映射后仍关于像曲线对称 |

!!! warning "保圆性中的"圆"包括直线"
    - 若圆周上**没有点**映射为 $\infty$ → 映射为有限半径的圆
    - 若圆周上**有一个点**映射为 $\infty$ → 映射为直线

### 6.2.3 唯一性定理

给定 $z$ 平面上三个相异点 $z_1, z_2, z_3$ 和 $w$ 平面上三个相异点 $w_1, w_2, w_3$，存在**唯一**的分式线性映射将 $z_k \mapsto w_k$：

$$\frac{w - w_1}{w - w_2} : \frac{w_3 - w_1}{w_3 - w_2} = \frac{z - z_1}{z - z_2} : \frac{z_3 - z_1}{z_3 - z_2}$$

### 6.2.4 两个重要的分式线性映射

**(1) 上半平面 → 单位圆内部**：
$$w = e^{i\theta}\frac{z - \lambda}{z - \overline{\lambda}}, \quad \text{Im}(\lambda) > 0$$
将 $\lambda$ 映射为圆心。

**(2) 单位圆内部 → 单位圆内部**：
$$w = e^{i\theta}\frac{z - z_0}{1 - \overline{z_0}z}, \quad |z_0| < 1$$
将 $z_0$ 映射为圆心。

### 6.2.5 区域映射的判断

| 圆周上的点映射情况 | 区域的像 |
|-------------------|---------|
| 没有点映射为 $\infty$ | 两圆弧围成的区域 |
| 一个点映射为 $\infty$ | 圆弧与直线围成的区域 |
| 一个交点映射为 $\infty$ | 角形区域 |

!!! example "将月牙域映射为角形域"
    将两圆弧交点 $i \mapsto 0$，$-i \mapsto \infty$，用 $w = k\frac{z-i}{z+i}$。
    再用第三个点确定常数 $k$ 即可。

## 6.3 初等函数的映射 — Mappings of Elementary Functions

### 6.3.1 幂函数 $w = z^n$（$n \geq 2$）

$$z = re^{i\theta} \xrightarrow{w=z^n} w = r^n e^{in\theta}$$

| 特性 | 说明 |
|------|------|
| 模 | $|w| = |z|^n$，圆周映射为圆周 |
| 辐角 | $\arg w = n\arg z$，射线映射为射线 |
| 角度 | 以原点为顶点的角形域，张角变为原来的 **$n$ 倍** |
| 一一对应 | 不是！在 $z = 0$ 处 $f'(0) = 0$，不保角 |

```
z 平面                    w 平面
  角形域 0 < θ < π/n        角形域 0 < φ < π
    \                          |
     \  n倍扩大                 |
      \  --------->            |
       \                       |
        \                      |
```

### 6.3.2 指数函数 $w = e^z$

$$z = x + iy \xrightarrow{w=e^z} w = e^x e^{iy}$$

| 特性 | 说明 |
|------|------|
| 水平带形域 $0 < \text{Im}(z) < a$ | 映射为角形域 $0 < \arg w < a$ |
| 竖直线 $x = c$ | 映射为圆周 $|w| = e^c$ |
| 水平线 $y = c$ | 映射为射线 $\arg w = c$ |
| 保角性 | 在全平面保角（$e^z \neq 0$） |
| 一一对应 | 在带形域 $0 < \text{Im}(z) < 2\pi$ 内一一 |

!!! example "指数函数的映射特点"
    把水平带形域 $0 < \text{Im}(z) < a$（$a \leq \pi$）映射成角形域 $0 < \arg w < a$。
    当 $a = \pi$ 时，上半平面映射为上半平面（除去实轴）。
