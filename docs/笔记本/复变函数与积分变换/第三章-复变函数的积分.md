# 第三章 复变函数的积分 — Complex Integration

!!! abstract
    本章研究沿复平面中曲线的积分。核心定理是 Cauchy 积分定理和 Cauchy 积分公式——它们是整个复变函数理论的基石，由此推出解析函数的无穷次可导性、Liouville 定理、最大模原理等重要结论。

## 3.1 复积分的定义 — Definition of Complex Integrals

### 3.1.1 Riemann 积分

设有向曲线 $C: z = z(t)$（$\alpha \leq t \leq \beta$），$f(z)$ 在 $C$ 上有定义。将 $C$ 分割为 $n$ 段，取 $\zeta_k$ 为第 $k$ 段上任意一点，作 Riemann 和：

$$S_n = \sum_{k=1}^{n} f(\zeta_k) \Delta z_k, \quad \Delta z_k = z_k - z_{k-1}$$

当最大弧长 $\delta \to 0$ 时，若极限存在，则定义：

$$\int_C f(z)\,dz = \lim_{\delta \to 0} \sum_{k=1}^{n} f(\zeta_k) \Delta z_k$$

### 3.1.2 计算方法

**定理**：$f(z) = u + iv$ 在 $C$ 上连续，则

$$\int_C f(z)\,dz = \int_C (u\,dx - v\,dy) + i \int_C (v\,dx + u\,dy)$$

**参数化法**：$z(t) = x(t) + iy(t)$（$\alpha \leq t \leq \beta$）

$$\int_C f(z)\,dz = \int_{\alpha}^{\beta} f(z(t)) \cdot z'(t)\,dt$$

!!! question "围道积分记号"
    - $\oint_C f(z)\,dz$ 表示沿**闭曲线** $C$ 的积分（围道积分）
    - $\oint_{|z-1|=1} f(z)\,dz$ 表示沿以 1 为圆心、1 为半径的圆周的积分

### 3.1.3 积分的性质

| 性质 | 公式 |
|------|------|
| 线性性 | $\int_C (af + bg)\,dz = a\int_C f\,dz + b\int_C g\,dz$ |
| 反向曲线 | $\int_{C^-} f\,dz = -\int_C f\,dz$ |
| 分段曲线 | $\int_{C_1 + C_2} f\,dz = \int_{C_1} f\,dz + \int_{C_2} f\,dz$ |
| 估值不等式 | $\left|\int_C f(z)\,dz\right| \leq \int_C |f(z)|\,|dz| \leq ML$ |

其中 $M = \max\limits_{z \in C} |f(z)|$，$L$ 为曲线 $C$ 的长度。

!!! example "计算 $\oint_C \frac{dz}{(z - z_0)^n}$，$C: |z - z_0| = r$"
    
    参数化：$z = z_0 + re^{i\theta}$，$dz = ire^{i\theta}d\theta$
    $$I = \int_0^{2\pi} \frac{ire^{i\theta}}{(re^{i\theta})^n}d\theta = \frac{i}{r^{n-1}}\int_0^{2\pi} e^{-i(n-1)\theta}d\theta$$
    $$I = \begin{cases} 2\pi i, & n = 1 \\ 0, & n \neq 1 \end{cases}$$

!!! example "积分估值"
    证明 $\left|\oint_C \frac{z+1}{z-1}dz\right| \leq 8\pi$，其中 $C: |z-1| = 2$。
    
    $$\left|\oint_C \frac{z+1}{z-1}dz\right| \leq \oint_C \left|\frac{z+1}{z-1}\right||dz| = \oint_C \frac{|z+1|}{2}|dz|$$
    由三角不等式：$|z+1| = |(z-1)+2| \leq |z-1| + 2 = 4$
    $$\leq \oint_C \frac{4}{2}|dz| = 2 \cdot (2\pi \cdot 2) = 8\pi$$

!!! warning "区分绝对值的位置"
    $$\oint_{|z|=1} \left|\frac{dz}{z}\right| = \oint_{|z|=1} |dz| = 2\pi$$
    $$\oint_{|z|=1} \frac{dz}{|z|} = \oint_{|z|=1} dz = 0$$
    $$\oint_{|z|=1} \frac{|dz|}{z} = \int_0^{2\pi} e^{-i\theta}d\theta = 0$$
    三者结果完全不同！

## 3.2 Cauchy 积分定理 — Cauchy's Integral Theorem

### 3.2.1 基本定理

**定理（Cauchy-Goursat）**：若 $f(z)$ 在单连通域 $D$ 内处处解析，$C$ 为 $D$ 内任意封闭曲线，则

$$\oint_C f(z)\,dz = 0$$

!!! warning
    曲线 $C$ 必须**完全属于**区域 $D$。如果 $f(z)$ 在 $C$ 内部有奇点，定理不成立。

### 3.2.2 推论

**路径无关性**：若 $f(z)$ 在单连通域 $D$ 内解析，则 $\int_C f(z)\,dz$ 与路径无关，仅与起点和终点有关。

### 3.2.3 复合闭路定理（复合路径 Cauchy 定理）

设 $C$ 为外边界，$C_1, C_2, \ldots, C_n$ 为 $C$ 内部的简单闭曲线（互不包含且互不相交），$f(z)$ 在它们围成的多连通区域内解析，则

$$\oint_C f(z)\,dz = \sum_{k=1}^{n} \oint_{C_k} f(z)\,dz$$

（所有曲线均取**正向**，即逆时针方向）

## 3.3 原函数与 Newton-Leibniz 公式

**定理**：若 $f(z)$ 在单连通域 $D$ 内解析，则 $F(z) = \int_{z_0}^{z} f(\zeta)\,d\zeta$ 在 $D$ 内解析，且 $F'(z) = f(z)$。

**Newton-Leibniz 公式**：若 $\Phi(z)$ 为 $f(z)$ 在 $D$ 内的原函数，则

$$\int_{z_0}^{z_1} f(\zeta)\,d\zeta = \Phi(z_1) - \Phi(z_0)$$

!!! example
    在区域 $D: -\pi < \arg z < \pi$ 内，$f(z) = \frac{1}{z}$ 的原函数为 $\ln z$：
    $$\int_1^z \frac{1}{\zeta}d\zeta = \ln z - \ln 1 = \ln z$$

---

## 3.4 Cauchy 积分公式 — Cauchy's Integral Formula

### 3.4.1 基本公式

**定理**：若 $f(z)$ 在区域 $D$ 内解析，$C$ 为 $D$ 内正向简单闭曲线，$z_0$ 在 $C$ 内部，则

$$f(z_0) = \frac{1}{2\pi i} \oint_C \frac{f(z)}{z - z_0}\,dz$$

**物理意义**：解析函数在内部任一点的值完全由它在边界上的值决定。

### 3.4.2 平均值公式

当 $C$ 为圆周 $z = z_0 + Re^{i\theta}$ 时：

$$f(z_0) = \frac{1}{2\pi}\int_0^{2\pi} f(z_0 + Re^{i\theta})\,d\theta$$

即：解析函数在圆心处的值等于它在圆周上的**平均值**。

### 3.4.3 高阶导数公式

**定理**：解析函数的 $n$ 阶导数为

$$f^{(n)}(z_0) = \frac{n!}{2\pi i} \oint_C \frac{f(z)}{(z - z_0)^{n+1}}\,dz$$

等价地：

$$\oint_C \frac{f(z)}{(z - z_0)^{n+1}}\,dz = \frac{2\pi i}{n!} f^{(n)}(z_0)$$

!!! question "这个公式有什么用？"
    不是用来通过积分求导数，而是**通过求导来计算积分**！遇到形如 $\oint \frac{f(z)}{(z-z_0)^{n+1}}dz$ 的积分，直接求 $f^{(n)}(z_0)$ 即可。

### 3.4.4 Cauchy 不等式

若 $f(z)$ 在 $|z - z_0| = R$ 内解析，在边界上连续，$M = \max\limits_{|z-z_0|=R} |f(z)|$，则

$$|f^{(n)}(z_0)| \leq \frac{n! \cdot M}{R^n}$$

## 3.5 重要定理 — Important Theorems

### 3.5.1 Liouville 定理

**定理**：全平面的有界解析函数必为常数。

**证明思路**：由 Cauchy 不等式 $|f'(z_0)| \leq \frac{M}{R}$，令 $R \to \infty$ 得 $f'(z_0) = 0$，故 $f$ 为常数。

### 3.5.2 最大模原理

**定理**：若 $f(z)$ 在有界区域 $D$ 内解析，在闭区域 $\overline{D}$ 上连续，则 $|f(z)|$ 的最大值在**边界** $\partial D$ 上取得。

!!! warning
    这意味着：非常值的解析函数的模**不可能在区域内部**取到最大值。
