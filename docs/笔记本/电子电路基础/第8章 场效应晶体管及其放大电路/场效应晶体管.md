理想状态下：$i_D = \frac{1}{2}u_nC_{oz} \frac{W}{L}(V_{GS}-V_{tn})^2$，计算直流电流时，我们只需要用到这个公式

实际情况中：$i_D = \frac{1}{2}u_nC_{oz} \frac{W}{L}(V_{GS}-V_{tn})^2(1+λV_{DS})$，这个公式在计算交流小信号模型时再用

λ称为沟道长度调制系数

λ的物理意义：$\lambda \propto \frac{\Delta i_D}{\Delta v_{DS}} = \frac{\partial i_D}{\partial v_{DS}}$（等于直流工作点处切线斜率）

小信号输出电阻：$r_o =( \frac{\partial i_D}{\partial V_{DS}})^{-1}= \frac{1}{λI_D}=\frac{V_A}{I_D}$
