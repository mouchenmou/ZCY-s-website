# 二、储能元件——动态元件

动态元件的v~i关系不能用简单的线性方程来描述，而要用微分的v~i关系来表征。
1.通过接入动态元件使电路实现某一特殊功能。
2.在高频电路中，一些实际元件不能用电阻性模型来表示。
